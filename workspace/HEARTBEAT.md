# ☀️ 每日早间新闻

**Cron 任务已配置：每日早上 7:00 自动执行**

**执行流程：**
1. 定时触发（cron: 0 7 * * *）
2. 抓取天气（深圳 + 广州）及新闻
3. 发送早安问候 + 天气 + 新闻到群里

---

## 🌤️ 天气报告要求（必须包含）

### 双城市天气（深圳 + 广州）

**深圳：福田区香蜜湖**  
**广州：天河区太古汇**

每城市包含：
- 当前温度
- 湿度（百分比）
- 风力（风速 + 风向）
- 最高温、最低温
- 穿衣推荐
- 是否需要带伞
- 温差提示

### 格式示例
```
🌤️ 【深圳】（福田区香蜜湖）
   🌡️ 当前：17.9°C
   💧 湿度：63%
   🌬️ 风力：13.9km/h（北风）
   📈 最高：23.1°C | 📉 最低：15.3°C
   👕 穿衣：薄外套或夹克，建议穿长袖+外套
   ☔ 带伞：建议带伞，早晚偏凉
   💡 温差7.8°C，建议分层穿搭！
```

---

## 📰 新闻要求

**至少列出 10 条新闻**

**数据源（使用 Chrome DevTools MCP）：**
1. 知乎热榜
2. 微博热搜
3. 其他可用源

**格式：**
```
🔥 【知乎热榜】
1. 标题1 (热度)
2. 标题2 (热度)
...

🔥 【微博热搜】
1. 标题1
2. 标题2
...
```

## 📍 位置

- 深圳（福田区香蜜湖）：纬度 22.54，经度 114.06
- 广州（天河区太古汇）：纬度 23.08，经度 113.25

**天气数据源：** Open-Meteo 天气 API
