# 🎁 情侣小电视制作教程（手把手版）

> 基于 ESP32 的桌面小电视，送给女朋友的完美礼物！

---

## 📋 项目概述

**项目名称**：情侣小电视 (MiniTV-for-couples)
**开源地址**：https://github.com/Commander-bao/MiniTV-for-couples
**主控芯片**：ESP32-WROOM-32E
**屏幕**：2.4寸 TFT 彩屏 (ILI9341驱动)
**开发环境**：VSCode + PlatformIO

### 🎮 功能列表
- 天气时钟
- 电量显示
- 纪念日倒计时
- 情侣互动（发送"想你啦"消息）
- 动画切换
- 亮度调节
- 快速配网

## 🔧 第二步：环境配置

### 1. 安装 VSCode
下载安装：https://code.visualstudio.com/

### 2. 安装 PlatformIO 插件
打开 VSCode → 左侧 Extensions → 搜索 "PlatformIO IDE" → 安装

### 3. 下载项目代码
```bash
git clone https://github.com/Commander-bao/MiniTV-for-couples
```
或直接下载 ZIP

### 4. 打开项目
VSCode → File → Open Folder → 选择下载的项目文件夹

---

## 🔩 第三步：硬件制作

### 1. PCB 打样
- 使用嘉立创 EDA 打开原理图和 PCB 文件
- 导出 Gerber 文件
- 在嘉立创免费打样（两层板）

### 2. 焊接元件
按原理图焊接以下元件：
- ESP32 模组
- 屏幕排线座
- 按钮开关
- 充电电路
- 锂电池接口

### 3. 3D 打印外壳
- 使用项目中的 STL 文件
- 可以在淘宝/嘉立创打印
- 建议使用白色 PLA 材质

---

## 📱 第四步：固件烧录

### 1. 配置 WiFi
在 `src/config.h` 中修改：
```cpp
#define WIFI_SSID "你的WiFi名称"
#define WIFI_PASSWORD "你的WiFi密码"
```

### 2. 配置城市代码
在 `src/config.h` 中修改：
```cpp
#define CITY_CODE "101280600"  // 深圳代码
```
其他城市代码查询：https://www.nowapi.com/api/weather

### 3. 编译上传
1. 连接 ESP32 到电脑（通过 USB 转 TTL）
2. 在 PlatformIO 中选择正确的 COM 口
3. 点击 "Upload" 按钮

---

## 💡 第五步：配网使用

### 首次配网
1. 打开小电视
2. 手机连接 WiFi
3. 扫描屏幕上的二维码或访问：192.168.4.1
4. 输入家中 WiFi 信息
5. 等待连接成功

### 情侣互动功能
- 左侧按钮：发送"想你啦"
- 右侧按钮：发送"打卡"
- 对方设备会显示消息

---

## ⚠️ 常见问题

### Q: 屏幕不亮
A: 检查排线是否插好，检查背光电压

### Q: WiFi 连接失败
A: 确认 WiFi 是 2.4G，不是 5G

### Q: 编译报错
A: 确保 PlatformIO 已正确安装 ESP32 开发板
