## 🔌 接线图 (ESP32 + 2.4寸 ST7789)

### 屏幕引脚连接

| 屏幕引脚 | ESP32 引脚 | 说明 |
|----------|------------|------|
| VCC | 3.3V | 电源 |
| GND | GND | 地 |
| SCL/SCK | GPIO18 | 时钟 |
| SDA/MOSI | GPIO23 | 数据 |
| RES | GPIO4 | 复位 |
| DC | GPIO2 | 数据/命令 |
| CS | GPIO5 | 片选 |
| BL | 3.3V 或 GPIO | 背光(可选) |

### 按钮连接

| 按钮 | ESP32 引脚 |
|------|------------|
| 按钮1 | GPIO15 |
| 按钮2 | GPIO16 |

## 🛠️ 第一步：安装 Arduino IDE

1. 下载 Arduino IDE：https://www.arduino.cc/en/software
2. 安装后打开 → 菜单：文件 → 首选项
3. 附加开发板管理器网址添加：
   ```
   https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
   ```
4. 工具 → 开发板 → 开发板管理器 → 搜索 "esp32" → 安装

---

## 📚 第二步：安装库

在 Arduino IDE 中：工具 → 管理库 → 安装以下库：
1. **TFT_eSPI** - 屏幕驱动
2. **PubSubClient** - MQTT通信(可选)
3. **ArduinoJson** - JSON解析
4. **Time** - 时间同步

---

## ⚙️ 第三步：配置 TFT_eSPI 库

找到 Arduino 库目录下的 `TFT_eSPI/User_Setup.h`，修改：

```cpp
#define ST7789_DRIVER

#define TFT_WIDTH  240
#define TFT_HEIGHT 320

#define TFT_MOSI  23
#define TFT_SCLK  18
#define TFT_CS    5
#define TFT_DC    2
#define TFT_RST   4

#define LOAD_GLCD   // 字体
#define LOAD_FONT2
```

## 💻 第四步：代码示例

这是一个基础框架，你可以基于此添加情侣互动功能：

```cpp
#include <TFT_eSPI.h>
#include <WiFi.h>
#include <time.h>

TFT_eSPI tft = TFT_eSPI();

const char* ssid = "你的WiFi";
const char* password = "WiFi密码";

int button1Pin = 15;
int button2Pin = 16;

void setup() {
  Serial.begin(115200);
  
  // 初始化屏幕
  tft.init();
  tft.setRotation(1);
  tft.fillScreen(TFT_BLACK);
  
  // 连接WiFi
  WiFi.begin(ssid, password);
  while (WiFi.status() != WL_CONNECTED) {
    delay(500);
    tft.drawString("Connecting...", 10, 10);
  }
  tft.fillScreen(TFT_BLACK);
  tft.drawString("WiFi OK!", 10, 10);
  
  // 配置时间
  configTime(8 * 3600, 0, "ntp.aliyun.com");
  
  // 按钮输入
  pinMode(button1Pin, INPUT_PULLUP);
  pinMode(button2Pin, INPUT_PULLUP);
}

void loop() {
  // 显示时间
  time_t now = time(nullptr);
  struct tm* tm_info = localtime(&now);
  char timeStr[20];
  strftime(timeStr, 20, "%H:%M:%S", tm_info);
  
  tft.drawString(timeStr, 80, 100);
  
  // 检测按钮
  if (digitalRead(button1Pin) == LOW) {
    tft.drawString("想你啦!    ", 60, 150);
    delay(200);
  }
  
  if (digitalRead(button2Pin) == LOW) {
    tft.drawString("打卡!      ", 60, 150);
    delay(200);
  }
  
  delay(100);
}
```

## 🎮 情侣互动功能思路

1. **MQTT 通信**：两台设备通过 MQTT 主题通信
2. **按键触发**：按下按钮发送消息到对方设备
3. **显示消息**：收到消息后显示在屏幕上
