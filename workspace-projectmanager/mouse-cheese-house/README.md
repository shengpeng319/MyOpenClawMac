# 🐭 小老鼠奶酪屋 - 减法学习游戏

## 🎮 游戏简介

跟大毛和小毛一起学习有趣的减法技巧！这是一个专为7岁儿童设计的两位数减法学习游戏。

## 🐭 角色介绍

- **🐭 大毛（拆大数法）** - 把被减数拆成整十+十几
  - 例：24-6 → 24=10+14 → 14-6=8 → 20+8=28
  
- **🐭 小毛（拆小数法）** - 从减数拆出凑十部分
  - 例：24-6 → 6=4+2 → 24-4=20 → 20-2=18

## 🎨 设计特色

- 森林风+奶酪屋主题
- 奶酪黄 #FFD93D、成功绿 #6BCB77 配色
- 圆润可爱的大字体
- 60px圆角按钮，适合儿童点击

## 🎬 动画效果

- 数字弹跳出现

- 答对撒花+星星飞舞
- 答错奶酪抖动+温柔鼓励
- 过关烟花+角色欢呼

## 📊 关卡设计

| 关卡 | 内容 |
|------|------|
| 1-2 | 拆大数法入门（11-30）|
| 3-4 | 拆大数法熟练（31-50）|
