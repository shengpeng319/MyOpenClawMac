# 记忆系统指南

_完整文档。启动时不读，需要时查阅。_

## 三层记忆

| 层 | 位置 | 内容 | 保留 |
|---|------|------|------|
| 工作 | Prompt 上下文 | 当前任务、最近对话 | 会话内 |
| 短期 | `memory/YYYY-MM-DD.md` | 今日对话、临时约定 | 7-30 天 |
| 长期 | `MEMORY.md`, `SOUL.md`, `USER.md` | 核心身份、用户画像、知识沉淀 | 持久 |

## 文件职责

- **MEMORY.md** — 精选记忆，主会话加载。决策、教训、偏好。
- **memory/YYYY-MM-DD.md** — 每日原始日志。捕获重要事件。
- **SOUL.md** — 身份与价值观，几乎不变。
- **USER.md** — 用户画像，缓慢演化。

## 管理规则

**写入时机：**
- 完成重要任务 → 记录到当天 memory 文件
- 发现用户新偏好 → 更新 USER.md
- 学到新技能 → 写入 *_BEST_PRACTICE.md 或 MEMORY.md
- 做出重大决策 → MEMORY.md

**每周心跳时：** 读本周 memory 文件，提炼 3-5 个关键点到 MEMORY.md。

**不要：**
- 依赖"脑子里记住了"
- 什么都记（过滤噪音）
- 重复存储同一信息

## Lossless-Claw 插件

- 上下文压缩时保留对话 DAG 结构
- 增量摘要，比默认更无损
- 配置位置：`~/.openclaw/lcm.db`

无需手动操作。
