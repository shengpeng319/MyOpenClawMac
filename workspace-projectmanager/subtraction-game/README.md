# 🐰 数学小兔子 - 两位数减法学习游戏

一个专为7岁儿童设计的两位数减法学习游戏，通过动画和互动帮助孩子掌握两种重要的减法技巧。

## 🎯 游戏特色

### 两种教学方法
1. **📦 拆大数法** - 把被减数（大数）拆分成整十数和个位数
2. **🎯 拆小数法** - 把减数（小数）拆分成整十数和个位数

### 难度等级
- 🌟 **简单** - 不需要借位的减法（如 65-23）
- 🌟🌟 **中等** - 简单借位（如 54-28）
- 🌟🌟🌟 **困难** - 复杂借位（如 72-37）

## ✨ 功能特点

- 🎨 可爱的卡通风格设计，适合儿童
- 🐰 小兔子动画，增加趣味性
- ⭐ 答题获得星星奖励系统
- 🎉 答对时的庆祝动画效果
- 📱 完全响应式设计，支持手机和平板
- 🎮 每轮10道题，结束后显示成绩统计

## 🚀 运行方式

直接在浏览器中打开 `index.html` 文件即可开始游戏：

```bash
# macOS
open subtraction-game/index.html

# Windows
start subtraction-game/index.html

# Linux
xdg-open subtraction-game/index.html
```

## 🎮 游戏规则

1. 选择难度等级
2. 点击"开始挑战"
3. 选择一种解题方法（拆大数法或拆小数法）
4. 观看解题步骤动画演示
5. 在输入框中输入答案
6. 点击"回答"或按回车提交
7. 每答对10题获得一颗星星
8. 完成10道题后查看成绩

## 📝 设计理念

本游戏基于儿童认知发展特点设计：
- 色彩鲜明，对比度高，易于识别
- 动画效果帮助理解抽象的数学概念
- 逐步展示计算过程，而非直接给出答案
- 正向激励系统，鼓励持续学习
