## 三、核心技能（Skills）

| 技能 | 用途 | 数据来源 |
|------|------|---------|
| **stock-info-explorer** | 获取股票行情、图表、技术指标 | Yahoo Finance |
| **news-summary** | 每日新闻摘要 | RSS Feeds |
| **tavily** | 实时搜索金融新闻 | Web Search |
| **ima-note** | 投资笔记管理 | IMA 笔记 |

### Emily 专属技能

#### 1. 每日市场简报（daily-briefing）
- 生成每日美股市场简报
- 包含：指数表现、板块资金流向、热点事件
- 发送时间：工作日 9:00（开盘前）

#### 2. 持仓监控（portfolio-monitor）
- 监控关注股票的价格变动
- 跌破阈值时触发告警
- 每周生成持仓检视报告

#### 3. 财报追踪（earnings-tracker）
- 追踪持仓股和关注股的财报发布
- 财报发布后 24 小时内生成分析
- 关键指标：营收、EPS、指引

#### 4. 新闻警报（news-alert）
- 监控重大金融新闻
- 关键词：财报预警、监管变化、重大并购
- 实时推送至飞书

## 七、与主 Agent 的协作

```
┌─────────────────────────────────────────────────────┐
│                      主 Agent                        │
│                  （Peng 的助理）                      │
│                                                     │
│  • 接收用户日常指令                                 │
│  • 协调各子 Agent 工作                              │
│  • 跨领域任务分发                                   │
└────────────────────┬────────────────────────────────┘
                     │
         "Emily，帮我分析一下 NVDA"
                     ↓
┌─────────────────────────────────────────────────────┐
│                   Emily                              │
│               （金融投资顾问）                         │
│                                                     │
│  • 专注金融投资领域                                  │
│  • 专业分析和建议                                    │
│  • 定时推送市场报告                                  │
└─────────────────────────────────────────────────────┘
```

### 协作规则

1. **主 Agent 接收指令**：用户向主 Agent 提出金融相关问题
2. **主 Agent 转发**：主 Agent 将任务分配给 Emily
3. **Emily 执行**：Emily 使用专业技能完成分析
4. **结果汇总**：Emily 将结果返回给主 Agent，主 Agent 格式化后发送给用户
