# 📰 每日热点新闻

**Cron 任务已配置：每日早上 8:00 自动执行**

**执行流程：**
1. 定时触发（cron: 0 8 * * *）
2. 发送系统事件消息
3. 你收到消息后说「开始抓取新闻」
4. 我打开 Chrome → 连接 Browser Relay → 抓取新闻

**来源：** 知乎、微博、虎嗅、36氪、观察者网、华尔街日报

# 📊 每日进化报告

**Cron 任务已配置：每日晚上 22:00 自动执行**

**执行流程：**
1. 定时触发（cron: 0 22 * * *）
2. 发送系统事件消息
3. 你收到消息后执行：
   - 回顾今天的会话历史
   - 提取关键要点
   - 分析学会了什么、犯了什么错、怎么解决的
   - 形成每日进化报告，提议可以固化的技能
4. 发送报告给用户
