## 飞书文件推送规范

### ✅ 正确方式
```python
# 1. 先上传到云盘
feishu_drive action=create_folder name="目标文件夹"
feishu_drive action=info file_token="xxx"

# 2. 获取 file_token 后推送
message action=send file_token="xxx" ...
```

### ❌ 禁止方式
- 直接发送本地文件路径
- 不验证直接推送

## 验证流程

```
生成文件

  ↓
上传云盘（获取file_token）
  ↓

  ↓
请用户确认
  ↓
完成 ✅
```
