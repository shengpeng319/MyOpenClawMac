### 什么是交易心理？

交易心理是量化交易的"隐形引擎"。即使你有完美的策略，如果心理不过关，市场只需要连续几次波动就能让你心态崩溃、纪律全无。

**类比：** 交易心理就像赛车的刹车。再快的速度，如果刹车失灵，一次失误就可能让你退出赛场。

### 1.1 损失厌恶 (Loss Aversion)

**定义：** 亏损的痛苦感是盈利快乐感的 2-3 倍。

**症状：**
- 盈利时急着落袋为安
- 亏损时死扛不卖
- "等回本就卖"成为口头禅

**解决方案：** 提前设定止损位，让规则代替情绪。

```python
import pandas as pd
import numpy as np

def check_loss_aversion(trade_df):
    """
    检测损失厌恶倾向
    
    参数:
        trade_df: DataFrame，需包含 pnl（盈亏金额）、hold_hours（持仓小时数）列
    
    返回:
        dict: 包含 ratio（亏损/盈利持仓时间比）、判断结果
    """
    # 输入验证
    required_cols = ['pnl', 'hold_hours']
    if not all(col in trade_df.columns for col in required_cols):
        raise ValueError(f"DataFrame 必须包含列: {required_cols}")
    
    # 分离盈利/亏损交易
    winning = trade_df[trade_df['pnl'] > 0]['hold_hours']
    losing = trade_df[trade_df['pnl'] <= 0]['hold_hours']
    
    # 计算平均持仓时间
    avg_winning_hours = winning.mean() if len(winning) > 0 else 0
    avg_losing_hours = losing.mean() if len(losing) > 0 else 0
    
    # 亏损持仓时间/盈利持仓时间（避免除零）
    if avg_winning_hours > 0:
        ratio = avg_losing_hours / avg_winning_hours
    else:
        ratio = float('inf') if avg_losing_hours > 0 else 0
    
    print(f"盈利交易数: {len(winning)}, 平均持仓: {avg_winning_hours:.1f} 小时")
    print(f"亏损交易数: {len(losing)}, 平均持仓: {avg_losing_hours:.1f} 小时")
    print(f"亏损持仓时间/盈利持仓时间 = {ratio:.2f}")
    
    if ratio > 2:
        print("⚠️ 检测到损失厌恶倾向！亏损持仓时间显著过长")
        return {'ratio': ratio, 'detected': True, 'severity': 'high'}
    elif ratio > 1.5:
        print("🟡 轻度损失厌恶倾向，建议关注")
        return {'ratio': ratio, 'detected': True, 'severity': 'medium'}
    else:
        print("✅ 持仓时间比例正常")
        return {'ratio': ratio, 'detected': False, 'severity': 'low'}
```

### 1.2 过度自信 (Overconfidence)

**定义：** 高估自己的分析能力，低估市场风险。

**症状：**
- 连续盈利后加大仓位
- 不做风控，全仓进出
- 认为"我比市场聪明"

**解决方案：** 用数据说话，定期做策略回顾，不要因为短期结果改变长期规则。

```python
def check_overconfidence(trade_df):
    """
    检测过度自信倾向
    
    检查维度:
    1. 仓位变化（前半段 vs 后半段）
    2. 交易频率变化
    3. 仓位与频率变化的综合评估
    
    参数:
        trade_df: DataFrame，需包含 timestamp、position_size 列
    
    返回:
        dict: 各维度检测结果及综合判断
    """
    # 输入验证
    required_cols = ['timestamp', 'position_size']
    if not all(col in trade_df.columns for col in required_cols):
        raise ValueError(f"DataFrame 必须包含列: {required_cols}")
    
    # 按时间排序
    df = trade_df.sort_values('timestamp').reset_index(drop=True)
    n = len(df)
    
    if n < 4:
        return {'error': '数据量不足（至少需要4笔交易）'}
    
    mid = n // 2
    first_half = df.iloc[:mid]
    second_half = df.iloc[mid:]
    
    results = {}
    
    # === 维度1: 仓位变化检测 ===
    avg_pos_first = first_half['position_size'].mean()
    avg_pos_second = second_half['position_size'].mean()
    
    # 防止除零：仓位为0时设为1来计算百分比变化
    pos_base = avg_pos_first if avg_pos_first != 0 else 1
    pos_change = (avg_pos_second - avg_pos_first) / pos_base
    
    results['position_change'] = pos_change
    results['avg_position_first_half'] = avg_pos_first
    results['avg_position_second_half'] = avg_pos_second
    
    print(f"【仓位分析】")
    print(f"  前半段平均仓位: {avg_pos_first:.0f}")
    print(f"  后半段平均仓位: {avg_pos_second:.0f}")
    print(f"  仓位变化: {pos_change*100:+.1f}%")
    
    # === 维度2: 交易频率变化 ===
    # 按天统计交易次数
    df['date'] = pd.to_datetime(df['timestamp']).dt.date
    daily_counts = df.groupby('date').size()
    
    first_half_days = df.iloc[:mid]['date'].nunique()
    second_half_days = df.iloc[mid:]['date'].nunique()
    
    freq_first = len(first_half) / max(first_half_days, 1)
    freq_second = len(second_half) / max(second_half_days, 1)
    freq_change = (freq_second - freq_first) / max(freq_first, 0.1)
    
    results['frequency_change'] = freq_change
    
    print(f"【频率分析】")
    print(f"  前半段日均交易: {freq_first:.2f} 笔")
    print(f"  后半段日均交易: {freq_second:.2f} 笔")
    print(f"  频率变化: {freq_change*100:+.1f}%")
    
    # === 综合判断 ===
    detected = False
    severity = 'low'
    
    if pos_change > 0.5 or freq_change > 0.5:
        detected = True
        severity = 'high'
        print("⚠️ 检测到过度自信！后半段仓位或频率显著增加")
    elif pos_change > 0.3 or freq_change > 0.3:
        detected = True
        severity = 'medium'
        print("🟡 存在过度自信倾向，后半段交易行为趋于激进")
    else:
        print("✅ 交易行为无明显过度自信迹象")
    
    results['detected'] = detected
    results['severity'] = severity
    
    return results
```

### 1.3 确认偏误 (Confirmation Bias)

**定义：** 只看支持自己观点的信息，忽视相反证据。

**症状：**
- 只关注利好消息
- 亏损后怪"市场错了"
- 拒绝承认策略失效

**解决方案：** 定期做"反向思考"练习，主动寻找否定自己观点的证据。

### 2.1 交易前检查清单

每次下单前必须确认：

```
□ 符合策略信号吗？（RSI/MA/MACD 条件满足）
□ 止损位设定了吗？（ATR 或固定比例）
□ 单笔风险 < 2% 账户吗？
□ 仓位计算正确吗？
□ 今天不是重大新闻日吗？
□ 心情平稳吗？
□ 手机不是只剩 20% 电吗？（开玩笑，但真的重要）
```

### 2.2 交易后检查清单

每次平仓后必须记录：

```
□ 入场时间、价格、理由
□ 止损设置
□ 出场时间、价格、原因
□ 盈亏金额
□ 是否遵守规则？
□ 如果违背了规则，原因是什么？
□ 学到了什么？
```

### 3.1 为什么需要交易日志？

**数据证明：**

### 3.2 交易日志类

```python
import pandas as pd
from datetime import datetime
from typing import Optional, Dict, Any, List

class TradingJournal:
    """
    交易日志系统
    记录每一笔交易的心理状态和规则遵守情况
    
    使用方式:
        journal = TradingJournal()
        journal.log_trade(entry_time='2026-03-28 10:00', entry_price=150.00, ...)
        report = journal.generate_report()
    """
    
    def __init__(self):
        """初始化交易日志"""
        self.trades: List[Dict[str, Any]] = []
    
    def log_trade(self, **trade_info) -> None:
        """
        记录一笔交易
        
        参数:
            entry_time: 入场时间字符串
            entry_price: 入场价格
            ticker: 股票代码
            signal_type: 信号类型（如 'MA金叉'）
            position_size: 持仓数量
            stop_loss: 止损价格
            exit_time: 出场时间字符串
            exit_price: 出场价格
            pnl: 盈亏金额
            pre_trade_mood: 交易前心情 (1-10)
            post_trade_mood: 交易后心情 (1-10)
            rules_followed: 是否遵守规则
            deviation_reason: 违背规则的原因（可选）
            lessons_learned: 学到的教训
        """
        trade_record = {
            'timestamp': datetime.now(),
            'entry_time': trade_info.get('entry_time'),
            'entry_price': trade_info.get('entry_price'),
            'ticker': trade_info.get('ticker'),
            'signal_type': trade_info.get('signal_type'),
            'position_size': trade_info.get('position_size', 0),
            'stop_loss': trade_info.get('stop_loss'),
            'exit_time': trade_info.get('exit_time'),
            'exit_price': trade_info.get('exit_price'),
            'pnl': trade_info.get('pnl', 0),
            'pre_trade_mood': trade_info.get('pre_trade_mood'),
            'post_trade_mood': trade_info.get('post_trade_mood'),
            'rules_followed': trade_info.get('rules_followed', True),
            'deviation_reason': trade_info.get('deviation_reason'),
            'lessons_learned': trade_info.get('lessons_learned', ''),
        }
        self.trades.append(trade_record)
    
    def generate_report(self) -> pd.DataFrame:
        """
        生成交易心理分析报告
        
        返回:
            DataFrame: 所有交易记录
        """
        if not self.trades:
            print("⚠️ 日志为空，无交易记录可分析")
            return pd.DataFrame()
        
        df = pd.DataFrame(self.trades)
        
        print("=" * 50)
        print("交易心理分析报告")
        print("=" * 50)
        
        # === 规则遵守分析 ===
        rules_kept = df[df['rules_followed'] == True]['pnl']
        rules_broken = df[df['rules_followed'] == False]['pnl']
        
        print(f"\n【规则遵守分析】")
        print(f"遵守规则交易数: {len(rules_kept)} ({len(rules_kept)/len(df)*100:.1f}%)")
        
        if len(rules_kept) > 0:
            print(f"遵守规则平均盈利: ${rules_kept.mean():.2f}")
        else:
            print("遵守规则平均盈利: 无数据")
        
        if len(rules_broken) > 0:
            print(f"违背规则平均盈利: ${rules_broken.mean():.2f}")
        else:
            print("违背规则平均盈利: 无数据（说明全部遵守规则）")
        
        # === 心情与盈亏关系 ===
        if 'pre_trade_mood' in df.columns and df['pre_trade_mood'].notna().any():
            print(f"\n【心情与盈亏关系】")
            mood_groups = df.groupby('pre_trade_mood')['pnl'].agg(['mean', 'count'])
            mood_groups.columns = ['平均盈亏', '交易数']
            print(mood_groups.to_string())
        
        # === 盈亏统计 ===
        print(f"\n【盈亏统计】")
        print(f"总交易数: {len(df)}")
        print(f"盈利交易: {(df['pnl'] > 0).sum()}")
        print(f"亏损交易: {(df['pnl'] <= 0).sum()}")
        print(f"胜率: {(df['pnl'] > 0).mean()*100:.1f}%")
        print(f"总盈亏: ${df['pnl'].sum():.2f}")
        
        return df
    
    def detect_emotional_patterns(self) -> Dict[str, Any]:
        """
        检测情绪化交易模式
        
        分析两种情绪化模式:
        1. 亏损后更容易违背规则（报复性交易）
        2. 盈利后更容易违背规则（过度自信）
        
        返回:
            dict: 包含检测结果和建议
        """
        if len(self.trades) < 3:
            return {'error': '数据不足，需要至少3笔交易'}
        
        df = pd.DataFrame(self.trades)
        df = df.sort_values('timestamp').reset_index(drop=True)
        
        # 前一笔盈亏
        df['prev_pnl'] = df['pnl'].shift(1)
        
        # 过滤有效数据
        valid = df.dropna(subset=['prev_pnl'])
        
        if len(valid) == 0:
            return {'error': '无法计算前一盈亏'}
        
        after_loss = valid[valid['prev_pnl'] < 0]
        after_win = valid[valid['prev_pnl'] > 0]
        
        results = {}
        
        print("=" * 50)
        print("情绪化交易检测")
        print("=" * 50)
        
        # === 模式1: 亏损后情绪化 ===
        if len(after_loss) > 0:
            loss_deviation_rate = (after_loss['rules_followed'] == False).mean()
            print(f"亏损后违背规则比例: {loss_deviation_rate*100:.1f}%")
            results['loss_deviation_rate'] = loss_deviation_rate
        else:
            print("亏损后违背规则: 无数据")
            results['loss_deviation_rate'] = None
        
        # === 模式2: 盈利后情绪化 ===
        if len(after_win) > 0:
            win_deviation_rate = (after_win['rules_followed'] == False).mean()
            print(f"盈利后违背规则比例: {win_deviation_rate*100:.1f}%")
            results['win_deviation_rate'] = win_deviation_rate
        else:
            print("盈利后违背规则: 无数据")
            results['win_deviation_rate'] = None
        
        # === 综合判断 ===
        loss_rate = results.get('loss_deviation_rate')
        win_rate = results.get('win_deviation_rate')
        
        if loss_rate is not None and win_rate is not None:
            # 计算差异
            diff = abs(loss_rate - win_rate)
            
            if loss_rate > win_rate + 0.1:
                print("\n⚠️ 检测到 '亏损后更容易情绪化' 模式（报复性交易）")
                print("建议: 亏损后设置强制冷却期（至少30分钟不交易）")
                results['pattern_detected'] = 'loss'
            elif win_rate > loss_rate + 0.1:
                print("\n⚠️ 检测到 '盈利后更容易情绪化' 模式（过度自信）")
                print("建议: 盈利后适当降低仓位，避免盲目自信")
                results['pattern_detected'] = 'win'
            else:
                print("\n✅ 未检测到明显的情绪化交易模式")
                results['pattern_detected'] = False
        elif loss_rate is not None:
            if loss_rate > 0.3:
                print("\n⚠️ 亏损后违背规则比例偏高")
                results['pattern_detected'] = 'loss'
            else:
                results['pattern_detected'] = False
        elif win_rate is not None:
            if win_rate > 0.3:
                print("\n⚠️ 盈利后违背规则比例偏高")
                results['pattern_detected'] = 'win'
            else:
                results['pattern_detected'] = False
        else:
            results['pattern_detected'] = False
        
        return results


# === 使用示例 ===
if __name__ == "__main__":
```

```python
def analyze_your_psychological_biases(trade_df):
    """
    综合分析交易心理偏差
    
    参数:
        trade_df: DataFrame，需包含列:
            - pnl: 盈亏金额
            - hold_hours: 持仓小时数
            - timestamp: 交易时间
            - position_size: 持仓数量
            - rules_followed: 是否遵守规则
    
    返回:
        dict: 包含各维度分析结果
    """
    print("=" * 50)
    print("交易心理全面分析报告")
    print("=" * 50)
    
    results = {}
    
    # 1. 损失厌恶检测
    print("\n【1. 损失厌恶分析】")
    loss_aversion_result = check_loss_aversion(trade_df)
    results['loss_aversion'] = loss_aversion_result
    
    # 2. 过度自信检测
    print("\n【2. 过度自信分析】")
    overconfidence_result = check_overconfidence(trade_df)
    results['overconfidence'] = overconfidence_result
    
    # 3. 交易频率分析
    print("\n【3. 交易频率分析】")
    trade_df = trade_df.copy()
    trade_df['date'] = pd.to_datetime(trade_df['timestamp']).dt.date
    daily_counts = trade_df.groupby('date').size()
    
    avg_daily = daily_counts.mean()
    max_daily = daily_counts.max()
    
    print(f"日均交易次数: {avg_daily:.2f}")
    print(f"最多一天交易: {max_daily} 次")
    
    if max_daily > 10:
        print("⚠️ 交易过于频繁，可能受情绪驱动")
        results['frequency_alert'] = True
    else:
        print("✅ 交易频率正常")
        results['frequency_alert'] = False
    
    results['daily_frequency'] = avg_daily
    results['max_daily_frequency'] = max_daily
    
    # 4. 时段分析（修复：移除错误的 return 语句）
    print("\n【4. 交易时段分析】")
    trade_df['hour'] = pd.to_datetime(trade_df['timestamp']).dt.hour
    hourly_pnl = trade_df.groupby('hour')['pnl'].mean()
    
    print("各时段平均盈亏:")
    print(hourly_pnl.to_string())
    
    if not hourly_pnl.empty:
        worst_hour = hourly_pnl.idxmin()
        best_hour = hourly_pnl.idxmax()
        
        print(f"\n表现最差的时段: {worst_hour}:00 (平均 ${hourly_pnl[worst_hour]:.2f})")
        print(f"表现最好的时段: {best_hour}:00 (平均 ${hourly_pnl[best_hour]:.2f})")
        print("建议: 表现差的时段避免交易或减少仓位")
        
        results['worst_hour'] = worst_hour
        results['best_hour'] = best_hour
        results['hourly_pnl'] = hourly_pnl.to_dict()
    else:
        print("无足够数据进行分析")
        results['worst_hour'] = None
        results['best_hour'] = None
    
    # 5. 综合评分
    print("\n【5. 综合评分】")
    score = 100
    
    if loss_aversion_result.get('detected'):
        score -= 20 if loss_aversion_result.get('severity') == 'high' else 10
    
    if overconfidence_result.get('detected'):
        score -= 20 if overconfidence_result.get('severity') == 'high' else 10
    
    if results.get('frequency_alert'):
        score -= 10
    
    score = max(0, score)
    
    print(f"心理偏差综合评分: {score}/100")
    
    if score >= 80:
        print("✅ 交易心理状态良好")
    elif score >= 60:
        print("🟡 存在一定心理偏差，建议改进")
    else:
        print("⚠️ 心理偏差严重，建议暂停交易并系统学习")
    
    results['overall_score'] = score
    
    return results
```

### 练习 2：建立你的交易纪律清单

根据你的策略，回答以下问题并创建个性化清单：

### 5.2 冷静期规则

```python
COOLDOWN_RULES = """
【强制冷静期触发条件】
❌ 连续亏损 2 笔 → 强制休息 30 分钟

❌ 连续盈利 3 笔 → 强制休息 15 分钟（防止过度自信）
```

### 5.3 盈利后的陷阱

很多人不知道：**盈利后才是最大考验**

| 盈利后常见错误 | 正确做法 |
|--------------|---------|
| 盈利了，加大仓位 | 保持原仓位，让利润奔跑 |
| 觉得自己找到了圣杯 | 保持谦虚，继续按规则交易 |
| 想快速赚更多 | 按计划止盈，不要改变目标 |
| 看不起那些"没赚这么多"的人 | 专注于自己的系统 |
