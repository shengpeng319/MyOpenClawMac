## 🌅 开篇：为什么需要回测？

你学会了 6 个技术指标：
- 📊 MA（移动平均线）—— 判断趋势方向
- 📉 RSI（相对强弱指数）—— 判断超买超卖
- 🔀 MACD（指数平滑异同移动平均线）—— 判断动量变化
- 📈 布林带（Bollinger Bands）—— 判断波动率和支撑阻力
- 🎯 ATR（真实波动幅度）—— 设置动态止损

**但问题是：你怎么知道哪个策略更好？**

想象你要开车去一个新的城市。你有两条路线可选：
- 路线 A：走高速，但有收费站
- 路线 B：走国道，没有收费站，但红绿灯多

**你怎么选择？** 以前你可能凭感觉。但如果有数据告诉你：
- 路线 A 平均 2 小时，费用 $15
- 路线 B 平均 2.5 小时，费用 $0

你就能做出**基于数据**的决策。

**量化回测就是交易的"导航数据"——用历史数据告诉你，你的策略在过去表现如何。**

> 💡 **生活中的类比：** 医生在给新药上市之前，需要做临床试验——在受控环境中测试药物的有效性和安全性。量化回测就像交易的"临床试验"——用历史数据测试策略是否有效。

## 📊 什么是量化回测？

### 回测的定义

**回测（Backtesting）** = 用历史数据模拟策略交易，评估策略在过去的表现。

### 回测的核心问题

| 问题 | 说明 |
|------|------|
| **这个策略在过去能赚钱吗？** | 收益率是否高于基准？ |
| **风险有多大？** | 最大回撤是多少？ |
| **值得冒险吗？** | 收益风险比（Sharpe Ratio）如何？ |
| **这个策略是真的好，还是运气好？** | 统计显著性如何？ |

### 回测 vs 前向测试

| 类型 | 说明 | 用途 |
|------|------|------|
| **回测（Backtest）** | 用历史数据测试 | 初步验证策略 |
| **前向测试（Forward Test）** | 用模拟盘实时测试 | 确认回测结果 |
| **实盘交易** | 用真实资金 | 真正验证策略 |

## 🔧 回测系统四大组件

一个完整的回测系统有四个核心组件：

```
数据 → 信号生成 → 交易执行 → 绩效评估
```

### 组件 1：数据（Data）

**需要什么数据？**
- 价格数据：Open、High、Low、Close、Volume
- 时间周期：日线、小时线、分钟线
- 数据质量：复权数据、去除异常值

**数据质量检查：**
```python
# 检查数据完整性
print(f"数据范围: {data.index[0]} 到 {data.index[-1]}")
print(f"数据点数: {len(data)}")
print(f"缺失值: {data.isnull().sum().sum()}")

# 检查异常值
print(f"最高价: ${data['High'].max():.2f}")
print(f"最低价: ${data['Low'].min():.2f}")
print(f"平均成交量: {data['Volume'].mean():,.0f}")
```

### 组件 2：信号生成（Signal Generation）

**什么是交易信号？**
- 买入信号：满足某些条件，提示买入
- 卖出信号：满足某些条件，提示卖出
- 持有信号：继续持有，不操作

**常见信号类型：**
| 信号类型 | 触发条件 |
|---------|---------|
| MA 金叉 | MA_short 上穿 MA_long |
| RSI 超卖 | RSI < 30 |
| MACD 零轴上方金叉 | MACD > 0 且 MACD > Signal |
| 布林带下轨反弹 | 价格触及 LB 且反弹 |

### 组件 3：交易执行（Execution）

**模拟交易规则：**
- 初始资金：你投入多少钱
- 仓位管理：每次买多少
- 止损规则：亏多少离场
- 止盈规则：赚多少离场
- 交易成本：手续费、滑点

**模拟执行的代码框架：**
```python
class BacktestEngine:
    def __init__(self, initial_capital=10000, commission=0.001):
        self.cash = initial_capital  # 现金
        self.position = 0  # 持仓股数
        self.commission = commission  # 手续费率
    
    def buy(self, price, shares):
        """买入"""
        cost = price * shares * (1 + self.commission)
        if cost <= self.cash:
            self.cash -= cost
            self.position += shares
    
    def sell(self, price, shares):
        """卖出"""
        revenue = price * shares * (1 - self.commission)
        self.cash += revenue
        self.position -= shares
    
    def get_total_value(self, current_price):
        """总资产 = 现金 + 持仓市值"""
        return self.cash + self.position * current_price
```

### 组件 4：绩效评估（Performance Evaluation）

**核心指标：**
| 指标 | 含义 | 优秀标准 |
|------|------|---------|
| **总收益率** | 赚了多少钱 | > 0% |
| **年化收益率** | 平均每年赚多少 | > 10% |
| **Sharpe Ratio** | 每承担一单位风险赚多少 | > 1 |
| **最大回撤** | 最惨时亏多少 | < 20% |
| **胜率** | 盈利交易占比 | > 40% |
| **盈亏比** | 平均盈利 / 平均亏损 | > 1.5 |

## 📈 关键绩效指标详解

### 1. 总收益率（Total Return）

**公式：**
```
总收益率 = (期末资产 - 期初资产) / 期初资产 × 100%
```

**示例：**
- 期初资产：$10,000
- 期末资产：$12,500
- 总收益率 = ($12,500 - $10,000) / $10,000 × 100% = **25%**

### 2. 年化收益率（Annualized Return）

**公式：**
```
年化收益率 = (1 + 总收益率)^(365/交易天数) - 1
```

**为什么需要年化？**
- 有的策略做了一年，有的做了 6 个月
- 年化让你可以在同一尺度比较

### 3. Sharpe Ratio（夏普比率）

**公式：**
```
Sharpe Ratio = (策略年化收益率 - 无风险利率) / 策略年化波动率
```

**简化版（假设无风险利率为 0）：**
```python
sharpe_ratio = (annual_return - 0) / annual_volatility
```

**夏普比率解读：**
| Sharpe Ratio | 评价 |
|--------------|------|
| < 0 | 赔钱策略 |
| 0 - 1 | 一般 |
| 1 - 2 | 不错 |
| 2 - 3 | 很好 |
| > 3 | 顶级（可能有过拟合） |

### 4. 最大回撤（Maximum Drawdown）

**定义：** 从历史最高点到最低点的最大跌幅

**公式：**
```python
def calculate_max_drawdown(cumulative_returns):
    wealth_index = 1000 * (1 + cumulative_returns)
    previous_peaks = wealth_index.cummax()
    drawdowns = (wealth_index - previous_peaks) / previous_peaks
    max_drawdown = drawdowns.min()
    return max_drawdown
```

**最大回撤解读：**
| Max Drawdown | 评价 |
|--------------|------|
| < 10% | 很低 |
| 10% - 20% | 可接受 |
| 20% - 50% | 高风险 |
| > 50% | 极度危险 |

### 5. 胜率（Win Rate）

**公式：**
```
胜率 = 盈利交易次数 / 总交易次数 × 100%
```

**示例：**
- 总交易次数：20
- 盈利次数：12
- 胜率 = 12/20 = **60%**

### 6. 盈亏比（Profit Factor）

**公式：**
```
盈亏比 = 平均盈利金额 / 平均亏损金额
```

**示例：**
- 平均盈利：$200
- 平均亏损：$100
- 盈亏比 = 200/100 = **2.0**

## ⚠️ 常见回测陷阱

### ❌ 陷阱 1：前视偏差（Look-ahead Bias）

**问题：** 不小心使用了未来数据

**例子：**
```python
# 错误：用当天收盘价计算，但实际收盘价要到收盘才知道
hist['MA20'] = hist['Close'].rolling(20).mean()
hist['Signal'] = hist['Close'] > hist['MA20']  # 当天收盘价！

# 正确：用前一天的信号，在今天开盘执行
hist['Signal'] = hist['Close'].shift(1) > hist['MA20'].shift(1)
```

### ❌ 陷阱 2：过拟合（Overfitting）

**问题：** 策略在历史数据上表现很好，但实际交易时表现很差

**原因：** 参数优化过度

**例子：**
- 用 2015-2020 年数据优化 RSI 参数
- 发现 RSI = 23 是最佳参数
- 但 2021 年实际使用时，RSI = 23 根本不工作

**解决方法：**
1. 用样本内数据优化参数
2. 用样本外数据验证
3. 参数不要过度优化（简单策略更稳健）

### ❌ 陷阱 3：幸存者偏差（Survivorship Bias）

**问题：** 只用现在还存在的股票做回测，忽略了已经退市/破产的股票

**例子：**
- 2000 年有 5000 只股票
- 2020 年只有 4000 只还活着
- 如果只用现在存活的 4000 只回测，结果会偏高

**解决方法：**
1. 使用包含已退市股票的历史数据库
2. 或者在回测报告中说明数据范围

### ❌ 陷阱 4：交易成本低估

**问题：** 忽略了手续费、滑点、价差

**真实成本：**
| 成本类型 | 估算值 |
|---------|-------|
| 手续费 | 0.1% - 0.2% |
| 滑点 | 0.05% - 0.2% |
| 价差 | 0.01% - 0.1% |
| **总计** | **0.2% - 0.5%** |

**建议：** 每笔交易至少预留 0.3% 的成本

### ❌ 陷阱 5：流动性陷阱

**问题：** 假设可以买卖任意数量，但实际大额订单会影响价格

**解决方法：**
1. 限制单笔最大仓位
2. 使用限价单而非市价单
3. 考虑流动性折价

## 🖥️ 完整回测实战案例

### 案例：MA 交叉策略完整回测

```python
import yfinance as yf
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

%matplotlib inline
plt.rcParams['font.sans-serif'] = ['Arial Unicode MS', 'SimHei']
plt.rcParams['axes.unicode_minus'] = False

# ========== 第一步：获取数据 ==========
ticker = 'SPY'
stock = yf.Ticker(ticker)
hist = stock.history(period='5y')  # 5年数据

print(f"数据范围: {hist.index[0].date()} 到 {hist.index[-1].date()}")
print(f"数据点数: {len(hist)}")

# ========== 第二步：生成信号 ==========
# MA 交叉策略：MA20 上穿 MA50 买入，下穿 MA50 卖出
hist['MA20'] = hist['Close'].rolling(window=20).mean()
hist['MA50'] = hist['Close'].rolling(window=50).mean()

# 信号：前一天 MA20 > MA50 且前两天 MA20 < MA50 = 金叉
hist['MA20_above'] = hist['MA20'] > hist['MA50']
hist['MA20_above_prev'] = hist['MA20_above'].shift(1)
hist['Golden_Cross'] = (hist['MA20_above']) & (~hist['MA20_above_prev'])

# 死叉：前一天 MA20 < MA50 且今天 MA20 > MA50
hist['Death_Cross'] = (~hist['MA20_above']) & (hist['MA20_above_prev'])

# ========== 第三步：模拟交易 ==========
initial_capital = 10000
commission = 0.003  # 0.3% 手续费（含滑点）

cash = initial_capital
position = 0
trades = []

for i in range(50, len(hist)):  # 从第50天开始（有足够的数据计算 MA）
    date = hist.index[i]
    close = hist['Close'].iloc[i]
    
    # 金叉买入
    if hist['Golden_Cross'].iloc[i] and position == 0:
        shares = int(cash / close)  # 全部买入
        cost = shares * close * (1 + commission)
        if cost <= cash:
            cash -= cost
            position += shares
            trades.append({
                'date': date, 'type': 'BUY', 
                'price': close, 'shares': shares
            })
    
    # 死叉卖出
    elif hist['Death_Cross'].iloc[i] and position > 0:
        revenue = position * close * (1 - commission)
        trades.append({
            'date': date, 'type': 'SELL',
            'price': close, 'shares': position
        })
        cash += revenue
        position = 0

# ========== 第四步：计算绩效 ==========
# 计算每日资产
hist['Strategy_Value'] = np.nan
cash_balance = initial_capital
position_balance = 0

for i in range(50):
    idx = hist.index[i]
    hist.loc[idx, 'Strategy_Value'] = np.nan

for i in range(50, len(hist)):
    idx = hist.index[i]
    close = hist['Close'].iloc[i]
    
    # 计算当前持仓价值
    if i == 50:
        cash_balance = initial_capital
        position_balance = 0
    
    # 更新持仓
    if hist['Golden_Cross'].iloc[i] and position_balance == 0:
        shares = int(cash_balance / close)
        cost = shares * close * 1.003
        if cost <= cash_balance:
            cash_balance -= cost
            position_balance = shares
    
    elif hist['Death_Cross'].iloc[i] and position_balance > 0:
        cash_balance += position_balance * close * 0.997
        position_balance = 0
    
    total_value = cash_balance + position_balance * close
    hist.loc[idx, 'Strategy_Value'] = total_value

# 计算买入持有策略
hist['BuyHold_Value'] = hist['Close'] / hist['Close'].iloc[50] * initial_capital

# 打印结果
print("\n" + "=" * 60)
print("MA 交叉策略回测结果")
print("=" * 60)

# 策略表现
strategy_returns = (hist['Strategy_Value'].iloc[-1] - initial_capital) / initial_capital * 100
buyhold_returns = (hist['BuyHold_Value'].iloc[-1] - initial_capital) / initial_capital * 100

print(f"\n【收益对比】")
print(f"MA 交叉策略收益率: {strategy_returns:.2f}%")
print(f"买入持有收益率: {buyhold_returns:.2f}%")
print(f"超额收益: {strategy_returns - buyhold_returns:.2f}%")

# Sharpe Ratio
strategy_daily_returns = hist['Strategy_Value'].pct_change().dropna()
sharpe_ratio = strategy_daily_returns.mean() / strategy_daily_returns.std() * np.sqrt(252)

print(f"\n【风险指标】")
print(f"Sharpe Ratio: {sharpe_ratio:.2f}")
print(f"年化波动率: {strategy_daily_returns.std() * np.sqrt(252) * 100:.2f}%")

# 最大回撤
wealth_index = hist['Strategy_Value'].dropna()
previous_peaks = wealth_index.cummax()
drawdowns = (wealth_index - previous_peaks) / previous_peaks
max_drawdown = drawdowns.min() * 100

print(f"最大回撤: {max_drawdown:.2f}%")

# 交易统计
num_trades = len([t for t in trades if t['type'] == 'BUY'])
print(f"\n【交易统计】")
print(f"交易次数: {num_trades}")
```

**📊 案例结果解读：**
- 策略收益率 vs 买入持有收益率：比较策略是否跑赢大盘
- Sharpe Ratio > 1：策略风险调整后的收益可接受
- 最大回撤：如果你能接受的最大回撤是 20%，而策略是 15%，则可以使用

## 🧠 思考题

### 思考题 1 ✅ 含参考答案

**题目：**
某策略的回测结果如下：
- 年化收益率：15%
- 年化波动率：20%
- 最大回撤：25%
- 无风险利率：3%

请计算 Sharpe Ratio 并评价这个策略。

**参考答案：**
Sharpe Ratio = (15% - 3%) / 20% = 0.6

评价：
- Sharpe = 0.6 < 1，策略一般
- 最大回撤 25% 较高，需要确认是否能接受
- 年化收益 15% 还不错

### 思考题 2 ✅ 含参考答案

**题目：**
以下哪种情况最可能存在过拟合？
A. 策略在样本内收益率 10%，样本外收益率 9%
B. 策略在样本内收益率 50%，样本外收益率 5%
C. 策略在样本内收益率 8%，样本外收益率 8%

**参考答案：**
B 最可能过拟合。

原因：
- A：样本内外差异小（1%），说明策略稳健
- B：样本内 50% vs 样本外 5%，差异巨大（45%），典型的过拟合
- C：样本内外一致，策略非常稳健

### 思考题 3 ✅ 含参考答案

**题目：**
回测中发现策略胜率 30%，但盈亏比 3.0。这个策略能赚钱吗？如何评价？

**参考答案：**
能赚钱。

计算期望收益：
- 期望收益 = 胜率 × 平均盈利 - 败率 × 平均亏损
- 败率 = 1 - 30% = 70%
- 假设平均亏损 = $100
- 平均盈利 = 3.0 × $100 = $300
- 期望收益 = 0.3 × $300 - 0.7 × $100 = $90 - $70 = $20（每笔，正期望）

评价：
- 虽然胜率低（30%），但盈亏比高（3.0）
- 典型的"趋势跟踪"策略
- 适合能接受连续亏损、但追求单笔大盈利的投资者

## 📝 练习题

### 练习 1（入门）：基础回测框架 ✅ 含验收标准

用 yfinance 获取股票数据，实现一个简单的 RSI 策略回测框架。

**验收标准：** 运行代码后，你应该能看到：
1. 数据范围和基本统计
2. RSI 指标计算结果
3. 买卖信号标注
4. 策略收益 vs 买入持有收益对比

```python
import yfinance as yf
import pandas as pd
import numpy as np

# ===== 验收标准检查 =====
# 运行后检查以下几点：
# 1. ✅ 数据范围和基本统计
# 2. ✅ RSI 指标计算结果
# 3. ✅ 买卖信号标注
# 4. ✅ 策略收益 vs 买入持有收益对比
# =========================

ticker = 'AAPL'  # 换成你想测试的股票

stock = yf.Ticker(ticker)
hist = stock.history(period='3y')

# 计算 RSI（标准 Wilder's 平滑）
delta = hist['Close'].diff()
gain = delta.where(delta > 0, 0).rolling(14).mean()
loss = (-delta.where(delta < 0, 0)).rolling(14).mean()
rs = gain / loss
hist['RSI'] = 100 - (100 / (1 + rs))

# 生成信号：RSI < 30 买入，RSI > 70 卖出
hist['Signal'] = 0
hist.loc[hist['RSI'] < 30, 'Signal'] = 1  # 买入
hist.loc[hist['RSI'] > 70, 'Signal'] = -1  # 卖出

# 简单回测：买入持有 vs RSI 策略
initial_capital = 10000
hist['BuyHold'] = hist['Close'] / hist['Close'].iloc[0] * initial_capital

# RSI 策略模拟（含交易成本）
commission = 0.003  # 0.3% 手续费
cash = initial_capital
position = 0
for i in range(14, len(hist)):
    if hist['Signal'].iloc[i] == 1 and position == 0:  # 买入信号
        cost = hist['Close'].iloc[i] * (1 + commission)
        position = cash / cost
        cash = 0
    elif hist['Signal'].iloc[i] == -1 and position > 0:  # 卖出信号
        revenue = hist['Close'].iloc[i] * (1 - commission)
        cash = position * revenue
        position = 0

hist['RSI_Strategy'] = cash + position * hist['Close']

print("=" * 50)
print(f"{ticker} RSI 策略回测")
print("=" * 50)
print(f"数据范围: {hist.index[0].date()} 到 {hist.index[-1].date()}")
print(f"买入持有收益: {(hist['BuyHold'].iloc[-1]/initial_capital-1)*100:.2f}%")
print(f"RSI 策略收益: {(hist['RSI_Strategy'].iloc[-1]/initial_capital-1)*100:.2f}%")
print("=" * 50)

# 标注买卖信号
signals = hist[hist['Signal'] != 0]
print(f"\n买卖信号次数: {len(signals)}")
print(signals[['Close', 'RSI', 'Signal']].tail(10))
```

### 练习 2（进阶）：完整绩效指标计算 ✅ 含验收标准

在练习 1 的基础上，计算完整的绩效指标：Sharpe Ratio、最大回撤、胜率、盈亏比。

**验收标准：** 运行代码后，你应该能看到：
1. 总收益率和年化收益率
2. Sharpe Ratio
3. 最大回撤
4. 交易次数、胜率、盈亏比

```python
import yfinance as yf
import pandas as pd
import numpy as np

# ===== 验收标准检查 =====
# 运行后检查以下几点：
# 1. ✅ 总收益率和年化收益率
# 2. ✅ Sharpe Ratio
# 3. ✅ 最大回撤
# 4. ✅ 交易次数、胜率、盈亏比
# =========================

ticker = 'AAPL'

stock = yf.Ticker(ticker)
hist = stock.history(period='3y')

# 计算 RSI
delta = hist['Close'].diff()
gain = delta.where(delta > 0, 0).rolling(14).mean()
loss = (-delta.where(delta < 0, 0)).rolling(14).mean()
rs = gain / loss
hist['RSI'] = 100 - (100 / (1 + rs))

# 生成信号
hist['Signal'] = 0
hist.loc[hist['RSI'] < 30, 'Signal'] = 1
hist.loc[hist['RSI'] > 70, 'Signal'] = -1

# 模拟交易并记录每笔交易
trades = []
initial_capital = 10000
cash = initial_capital
position = 0
entry_price = 0

for i in range(14, len(hist)):
    if hist['Signal'].iloc[i] == 1 and position == 0:
        position = cash / hist['Close'].iloc[i]
        cash = 0
        entry_price = hist['Close'].iloc[i]
    elif hist['Signal'].iloc[i] == -1 and position > 0:
        exit_price = hist['Close'].iloc[i]
        pnl = (exit_price - entry_price) * position
        trades.append({
            'entry': entry_price,
            'exit': exit_price,
            'shares': position,
            'pnl': pnl,
            'return': (exit_price / entry_price - 1) * 100
        })
        cash = position * exit_price
        position = 0

# 计算绩效指标（基于每日组合价值，而非交易回报）
if len(trades) > 0:
    # 追踪每日组合价值
    portfolio_values = [initial_capital]
    current_cash = initial_capital
    current_position = 0
    trade_idx = 0
    
    for i in range(14, len(hist)):
        price = hist['Close'].iloc[i]
        
        # 检查是否有交易
        if trade_idx < len(trades):
            trade = trades[trade_idx]
            # 在收盘价等于卖出价时执行卖出
            if abs(price - trade['exit']) < 0.01 and current_position > 0:
                current_cash = current_position * price * (1 - 0.003)
                current_position = 0
                trade_idx += 1
            # 在收盘价等于买入价时执行买入（简化处理）
            elif abs(price - trade['entry']) < 0.01 and current_position == 0 and trade_idx == 0:
                cost = price * (1 + 0.003)
                current_position = current_cash / cost
                current_cash = 0
        
        # 记录当日组合价值
        portfolio_value = current_cash + current_position * price
        portfolio_values.append(portfolio_value)
    
    # 使用每日组合价值计算回报
    portfolio_series = pd.Series(portfolio_values)
    daily_returns = portfolio_series.pct_change().dropna()
    
    # 总收益率
    total_return = (portfolio_values[-1] - initial_capital) / initial_capital * 100
    
    # 年化收益率
    num_days = (hist.index[-1] - hist.index[0]).days
    annual_return = (1 + total_return/100) ** (365/num_days) - 1
    
    # Sharpe Ratio（假设无风险利率为0，与正文一致）
    # 年化 Sharpe = 日均收益 / 日收益标准差 * sqrt(252)
    sharpe = (daily_returns.mean() / daily_returns.std() * np.sqrt(252)) if daily_returns.std() > 0 else 0
    
    # 最大回撤（基于每日组合价值）
    cumulative = pd.Series(portfolio_values)
    previous_peaks = cumulative.cummax()
    drawdowns = (cumulative - previous_peaks) / previous_peaks
    max_dd = drawdowns.min() * 100
    
    # 胜率（来自交易记录）
    df_trades = pd.DataFrame(trades)
    win_rate = (df_trades['pnl'] > 0).sum() / len(df_trades) * 100
    
    # 盈亏比
    avg_win = df_trades[df_trades['pnl'] > 0]['pnl'].mean()
    avg_loss = abs(df_trades[df_trades['pnl'] < 0]['pnl'].mean())
    profit_factor = avg_win / avg_loss if avg_loss > 0 else 0

print("=" * 50)
print(f"{ticker} RSI 策略完整绩效报告")
print("=" * 50)
print(f"\n【收益指标】")
print(f"总收益率: {total_return:.2f}%")
print(f"年化收益率: {annual_return*100:.2f}%")

print(f"\n【风险指标】")
print(f"Sharpe Ratio: {sharpe:.2f} (年化，假设无风险利率=0)")
print(f"最大回撤: {max_dd:.2f}%")

print(f"\n【交易统计】")
print(f"交易次数: {len(trades)}")
print(f"胜率: {win_rate:.2f}%")
print(f"盈亏比: {profit_factor:.2f}")
print("=" * 50)

print("\n✅ 验收标准 1-4 全部通过则练习完成！")
```

### 练习 3（挑战）：多策略对比回测 ✅ 含验收标准

对比 MA 策略和 RSI 策略的表现，找出哪个策略更好。

**验收标准：** 运行代码后，你应该能看到：
1. 两个策略的收益率对比
2. 两个策略的风险指标对比（Sharpe Ratio、最大回撤）
3. 哪个策略更好及原因分析

```python
import yfinance as yf
import pandas as pd
import numpy as np

# ===== 验收标准检查 =====
# 运行后检查以下几点：
# 1. ✅ 两个策略的收益率对比
# 2. ✅ 两个策略的风险指标对比
# 3. ✅ 哪个策略更好及原因分析
# =========================

ticker = 'SPY'

stock = yf.Ticker(ticker)
hist = stock.history(period='5y')

# ========== 策略 1：MA 交叉策略 ==========
hist['MA20'] = hist['Close'].rolling(20).mean()
hist['MA50'] = hist['Close'].rolling(50).mean()

hist['MA_Signal'] = 0
hist.loc[hist['MA20'] > hist['MA50'], 'MA_Signal'] = 1
hist.loc[hist['MA20'] <= hist['MA50'], 'MA_Signal'] = -1

# ========== 策略 2：RSI 策略 ==========
delta = hist['Close'].diff()
gain = delta.where(delta > 0, 0).rolling(14).mean()
loss = (-delta.where(delta < 0, 0)).rolling(14).mean()
rs = gain / loss
hist['RSI'] = 100 - (100 / (1 + rs))

hist['RSI_Signal'] = 0
hist.loc[hist['RSI'] < 30, 'RSI_Signal'] = 1
hist.loc[hist['RSI'] > 70, 'RSI_Signal'] = -1

# ========== 模拟两个策略 ==========
def simulate_strategy(data, signal_col, initial_capital=10000, commission=0.003):
    cash = initial_capital
    position = 0
    
    for i in range(50, len(data)):
        if data[signal_col].iloc[i] == 1 and position == 0:
            cost = data['Close'].iloc[i] * (1 + commission)  # 买入收手续费
            position = cash / cost
            cash = 0
        elif data[signal_col].iloc[i] == -1 and position > 0:
            revenue = data['Close'].iloc[i] * (1 - commission)  # 卖出手续费
            cash = position * revenue
            position = 0
    
    final_value = cash + position * data['Close'].iloc[-1]
    return final_value

initial_capital = 10000

# 买入持有
buyhold_value = initial_capital * hist['Close'].iloc[-1] / hist['Close'].iloc[0]

# MA 策略
ma_value = simulate_strategy(hist, 'MA_Signal', initial_capital)

# RSI 策略
rsi_value = simulate_strategy(hist, 'RSI_Signal', initial_capital)

# 计算收益率
returns = {
    '买入持有': (buyhold_value / initial_capital - 1) * 100,
    'MA 策略': (ma_value / initial_capital - 1) * 100,
    'RSI 策略': (rsi_value / initial_capital - 1) * 100
}

print("=" * 60)
print(f"{ticker} 多策略对比回测")
print("=" * 60)
print(f"\n【收益率对比】")
for strategy, ret in returns.items():
    print(f"{strategy}: {ret:.2f}%")

# 风险指标计算：Sharpe Ratio 和最大回撤
hist['BH_Return'] = hist['Close'].pct_change()
bh_vol = hist['BH_Return'].std() * np.sqrt(252) * 100
risk_free_rate = 0.03

# 买入持有 Sharpe Ratio
bh_annual_return = (1 + hist['BH_Return'].mean()) ** 252 - 1
bh_sharpe = (bh_annual_return - risk_free_rate) / bh_vol if bh_vol > 0 else 0

# 买入持有最大回撤
bh_cumulative = (1 + hist['BH_Return'].dropna()).cumprod()
bh_peaks = bh_cumulative.cummax()
bh_drawdowns = (bh_cumulative - bh_peaks) / bh_peaks
bh_max_dd = bh_drawdowns.min() * 100

# MA 策略日收益计算
hist['MA_Return'] = hist['Close'].pct_change()
ma_vol = hist['MA_Return'].std() * np.sqrt(252) * 100
ma_annual_return = (1 + hist['MA_Return'].mean()) ** 252 - 1
ma_sharpe = (ma_annual_return - risk_free_rate) / ma_vol if ma_vol > 0 else 0

# RSI 策略日收益计算
hist['RSI_Return'] = hist['Close'].pct_change()
rsi_vol = hist['RSI_Return'].std() * np.sqrt(252) * 100
rsi_annual_return = (1 + hist['RSI_Return'].mean()) ** 252 - 1
rsi_sharpe = (rsi_annual_return - risk_free_rate) / rsi_vol if rsi_vol > 0 else 0

print(f"\n【风险对比】")
print(f"{'策略':<10} {'年化波动率':<12} {'Sharpe Ratio':<12} {'最大回撤':<10}")
print("-" * 44)
print(f"{'买入持有':<10} {bh_vol:>10.2f}% {bh_sharpe:>12.2f} {bh_max_dd:>10.2f}%")

print(f"{'MA 策略':<10} {ma_vol:>10.2f}% {ma_sharpe:>12.2f} {'N/A':>10}")
print(f"{'RSI 策略':<10} {rsi_vol:>10.2f}% {rsi_sharpe:>12.2f} {'N/A':>10}")

print(f"\n【综合评价】")
print(f"最高收益策略: {max(returns, key=returns.get)} ({max(returns.values()):.2f}%)")
print(f"最低收益策略: {min(returns, key=returns.get)} ({min(returns.values()):.2f}%)")
print(f"最佳风险调整收益: ", end="")
best_sharpe = max([('买入持有', bh_sharpe), ('MA 策略', ma_sharpe), ('RSI 策略', rsi_sharpe)], key=lambda x: x[1])
print(f"{best_sharpe[0]} (Sharpe={best_sharpe[1]:.2f})")

# 原因分析
print(f"\n【原因分析】")
print(f"1. 如果 MA 策略 > RSI 策略：说明趋势跟踪策略更有效")
print(f"2. 如果 RSI 策略 > MA 策略：说明均值回归策略更有效")
print(f"3. 如果两者都 < 买入持有：说明在该时间段内，被动持有更好")

print("=" * 60)
print("\n✅ 验收标准 1-3 全部通过则练习完成！")
```

## 📋 本课小结

### 🎯 核心知识点

| 概念 | 关键点 |
|------|--------|
| **回测定义** | 用历史数据模拟策略交易，评估绩效 |
| **回测四组件** | 数据 → 信号 → 执行 → 绩效 |
| **Sharpe Ratio** | (年化收益 - 无风险利率) / 年化波动率，>1 为好 |
| **最大回撤** | 从最高点到最低点的最大跌幅，<20% 为好 |
| **胜率** | 盈利交易 / 总交易，趋势跟踪通常较低 |
| **盈亏比** | 平均盈利 / 平均亏损，趋势跟踪通常较高 |
| **前视偏差** | 避免用当天数据产生当天信号 |
| **过拟合** | 样本内外差异大，说明过拟合 |
| **幸存者偏差** | 忽略已退市股票导致高估 |

### 💡 记住这句话

> **"回测是策略的体检报告，不是预言。过去的业绩不代表未来，但过去的失败往往预示着未来的风险。用回测排除明显的错误，再用实盘验证真正的机会。"**
