## 📖 第一部分：仓位计算方法

### 1.1 固定仓位比例法

**原理：** 每笔交易投入固定比例的资金。

```python
def fixed_position_size(capital, risk_pct, stop_loss_pct):
    """
    固定仓位比例计算
    
    参数:
        capital: 总资金
        risk_pct: 风险比例（默认1-2%）
        stop_loss_pct: 止损比例
    
    返回:
        仓位数量、实际风险金额
    """
    risk_amount = capital * (risk_pct / 100)  # 本次愿意承受的最大亏损
    position_size = risk_amount / stop_loss_pct  # 需要这么多市值才能用止损比例覆盖风险
    shares = int(position_size)  # 取整（股票至少1手=100股）
    
    actual_risk = shares * stop_loss_pct / 100
    return shares, actual_risk

# 示例
capital = 100000  # 10万资金
shares, risk = fixed_position_size(capital, risk_pct=1, stop_loss_pct=5)
print(f"买入股数: {shares}股")
print(f"实际风险: {risk:.2f}元")
```

**优点：** 简单易行，风险可控
**缺点：** 不能适应市场波动

### 1.2 ATR 动态仓位法

**原理：** 根据市场波动率动态调整仓位。波动大时少买，波动小时多买。

**ATR 获取方法：**
```python
import yfinance as yf
import pandas as pd

def get_atr(ticker, period='3mo'):
    """
    获取股票ATR指标
    
    参数:
        ticker: 股票代码（如 'AAPL'）
        period: 数据周期（建议3个月以上）
    
    返回:
        ATR值（美元）
    """
    data = yf.download(ticker, period=period, progress=False)
    high = data['High']
    low = data['Low']
    close = data['Close']
    
    # True Range 计算
    tr1 = high - low
    tr2 = abs(high - close.shift())
    tr3 = abs(low - close.shift())
    tr = pd.concat([tr1, tr2, tr3], axis=1).max(axis=1)
    
    # ATR = TR的14日简单移动平均
    atr = tr.rolling(14).mean().iloc[-1]
    return atr

# 使用示例
atr = get_atr('AAPL', period='3mo')
print(f"AAPL 14日ATR: ${atr:.2f}")
```

```python
def atr_position_size(capital, risk_pct, atr, current_price, atr_multiplier=2):
    """
    ATR动态仓位计算
    
    参数:
        capital: 总资金
        risk_pct: 风险比例（默认1-2%）
        atr: 平均真实波幅
        current_price: 当前价格
        atr_multiplier: ATR倍数（默认2倍止损）
    
    返回:
        (仓位数量, 止损价格, 风险金额)
    """
    risk_amount = capital * (risk_pct / 100)
    stop_distance = atr * atr_multiplier  # 止损距离 = 2倍ATR
    stop_price = current_price - stop_distance  # 止损价格
    position_value = risk_amount / (stop_distance / 100)  # 能承受的最大市值
    shares = int(position_value / current_price / 100) * 100  # 按手取整
    
    actual_risk = shares * stop_distance / 100
    return shares, stop_price, actual_risk

# ATR获取示例
import yfinance as yf

def get_atr(ticker, period='3mo'):
    """获取ATR指标（建议使用3个月数据，更稳定）"""
    data = yf.download(ticker, period=period, progress=False)
    high = data['High']
    low = data['Low']
    close = data['Close']
    
    tr1 = high - low
    tr2 = abs(high - close.shift())
    tr3 = abs(low - close.shift())
    tr = pd.concat([tr1, tr2, tr3], axis=1).max(axis=1)
    atr = tr.rolling(14).mean().iloc[-1]  # 14日ATR
    
    return atr

# 示例：计算AAPL的ATR动态仓位
ticker = 'AAPL'
atr = get_atr(ticker, period='3mo')  # 获取3个月的ATR
current_price = yf.download(ticker, period='1d', progress=False)['Close'].iloc[-1]

shares, stop_price, risk = atr_position_size(
    capital=100000,
    risk_pct=1,
    atr=atr,
    current_price=current_price,
    atr_multiplier=2
)

print(f"股票: {ticker}")
print(f"当前价格: ${current_price:.2f}")
print(f"ATR(14日): ${atr:.2f}")
print(f"止损距离: ${atr*2:.2f} ({((atr*2)/current_price*100):.2f}%)")
print(f"建议买入: {shares}股")
print(f"止损价格: ${stop_price:.2f}")
print(f"实际风险: ${risk:.2f}")
```

**优点：** 适应市场波动，自动调整
**缺点：** 需要实时 ATR 数据

### 1.3 凯利公式（Kelly Criterion）

**原理：** 根据胜率和盈亏比计算最优下注比例。

```python
def kelly_criterion(win_rate, avg_win, avg_loss):
    """
    凯利公式计算最优仓位
    
    参数:
        win_rate: 胜率（0-1）
        avg_win: 平均盈利
        avg_loss: 平均亏损（正数）
    
    返回:
        最优仓位比例（小数形式，建议取半凯利）
    
    公式: f* = (bp - q) / b
        b = 盈亏比 (avg_win / avg_loss)
        p = 胜率
        q = 1 - 胜率
    """
    if win_rate <= 0 or avg_loss <= 0:
        return 0
    
    b = avg_win / avg_loss  # 盈亏比
    q = 1 - win_rate
    p = win_rate
    
    kelly_pct = (b * p - q) / b
    
    # 实际使用建议取半凯利（f* / 2），降低波动
    half_kelly = kelly_pct / 2
    
    return max(0, min(kelly_pct, 1)), max(0, min(half_kelly, 0.5))

# 示例
win_rate = 0.55  # 55%胜率
avg_win = 1000   # 赚1000
avg_loss = 600   # 亏600

full_kelly, half_kelly = kelly_criterion(win_rate, avg_win, avg_loss)
print(f"全凯利仓位: {full_kelly*100:.1f}%")
print(f"半凯利仓位: {half_kelly*100:.1f}%")
print(f"建议仓位: {int(100000 * half_kelly / 100) * 100}元")
```

**警告：** 凯利公式假设收益分布稳定，实际交易中建议使用 **1/4 凯利或更小**，以应对市场变化和策略失效风险。

## 🛡️ 第二部分：风险管理核心原则

### 2.1 单笔最大亏损控制（2%原则）

**规则：** 任何单笔交易的最大亏损不超过总资金的 2%。

```python
def check_2pct_rule(capital, position_value, stop_loss_pct):
    """
    检查是否符合2%原则
    
    返回:
        (是否符合, 实际风险比例)
    """
    potential_loss = position_value * (stop_loss_pct / 100)
    risk_pct = (potential_loss / capital) * 100
    
    return risk_pct <= 2, risk_pct

# 示例
capital = 100000
position_value = 20000  # 想买入2万的股票
stop_loss = 5  # 5%止损

is_safe, risk_pct = check_2pct_rule(capital, position_value, stop_loss)
print(f"风险比例: {risk_pct:.2f}%")
print(f"是否符合2%原则: {'✅' if is_safe else '❌'}")
```

### 2.2 每日亏损上限

**规则：** 单日亏损达到 X% 时，停止交易。

```python
class DailyRiskManager:
    def __init__(self, capital, daily_limit_pct=3):
        self.initial_capital = capital
        self.daily_limit_pct = daily_limit_pct
        self.daily_pnl = 0
        self.trading_allowed = True
    
    def update(self, current_capital):
        """更新每日盈亏"""
        self.daily_pnl = current_capital - self.initial_capital
        daily_loss_pct = abs(self.daily_pnl / self.initial_capital * 100)
        
        if daily_loss_pct >= self.daily_limit_pct:
            self.trading_allowed = False
            print(f"⚠️ 今日亏损 {daily_loss_pct:.2f}%，停止交易")
        
        return self.trading_allowed
    
    def reset(self, new_capital):
        """新一天开始，重置"""
        self.initial_capital = new_capital
        self.daily_pnl = 0
        self.trading_allowed = True

# 使用示例
manager = DailyRiskManager(capital=100000, daily_limit_pct=3)
print(f"允许交易: {manager.trading_allowed}")
```

### 2.3 相关性管理与分散化

**规则：** 不要把所有仓位集中在相关度高的标的上。

```python
import numpy as np

def check_correlation_risk(positions, price_data):
    """
    检查持仓相关性风险
    
    返回:
        相关性矩阵、最高相关对
    """
    returns = price_data.pct_change().dropna()
    corr_matrix = returns.corr()
    
    # 找出高相关对（>0.7）
    high_corr = []
    for i in range(len(corr_matrix.columns)):
        for j in range(i+1, len(corr_matrix.columns)):
            if abs(corr_matrix.iloc[i, j]) > 0.7:
                high_corr.append((
                    corr_matrix.columns[i],
                    corr_matrix.columns[j],
                    corr_matrix.iloc[i, j]
                ))
    
    return corr_matrix, high_corr

# 示例：检查持仓相关性
symbols = ['AAPL', 'MSFT', 'GOOGL', 'AMZN', 'META']
data = yf.download(symbols, period='3mo')['Close']
corr, high_corr = check_correlation_risk(['AAPL', 'MSFT'], data)

print("高相关交易对:")
for s1, s2, corr_val in high_corr:
    print(f"  {s1} vs {s2}: {corr_val:.2f}")
```

### 案例1：10万资金，如何分配？

```python
def portfolio_allocation(capital, positions_config):
    """
    资金分配计算器
    
    positions_config: [(股票, 风险比例, 止损比例), ...]
    """
    results = []
    total_risk = 0
    
    for ticker, risk_pct, stop_pct in positions_config:
        risk_amount = capital * (risk_pct / 100)
        position_value = risk_amount / (stop_pct / 100)
        
        results.append({
            'ticker': ticker,
            'risk_amount': risk_amount,
            'position_value': position_value,
            'risk_pct': risk_pct
        })
        total_risk += risk_pct
    
    return results, total_risk

# 示例配置
config = [
    ('AAPL', 1.5, 5),   # 苹果：1.5%风险，5%止损
    ('MSFT', 1.0, 4),   # 微软：1%风险，4%止损
    ('GOOGL', 0.5, 6),  # 谷歌：0.5%风险，6%止损
]

capital = 100000
allocation, total = portfolio_allocation(capital, config)

print(f"总资金: {capital}元")
print(f"总风险暴露: {total}%")
print("\n各仓位:")
for pos in allocation:
    print(f"  {pos['ticker']}: 投入{pos['position_value']:.0f}元，风险{pos['risk_amount']:.0f}元({pos['risk_pct']}%)")
```

### 案例2：连续亏损后怎么办？

```python
def after_losing_streak(capital, losing_streak, base_risk_pct=1.0, min_risk_pct=0.5):
    """
    连续亏损后的仓位调整策略
    
    规则：每连续亏损2次，降低25%仓位；最低不低于0.5%
    """
    reduction_factor = 0.75  # 每次降低25%
    new_risk_pct = base_risk_pct * (reduction_factor ** (losing_streak // 2))
    new_risk_pct = max(new_risk_pct, min_risk_pct)
    
    return new_risk_pct

# 示例
print("连续亏损后的建议风险比例:")
for streak in [0, 2, 4, 6, 8]:
    risk = after_losing_streak(100000, streak)
    print(f"  连续亏损{streak}次: 风险比例 {risk:.2f}%")
```

### 4.1 自动化止损计算

```python
class AutoStopLoss:
    def __init__(self, initial_capital, max_risk_pct=2, daily_loss_limit_pct=3):
        self.capital = initial_capital
        self.max_risk_pct = max_risk_pct
        self.daily_loss_limit_pct = daily_loss_limit_pct
        self.positions = {}  # {ticker: {'shares': n, 'entry': price, 'stop': price}}
        self.daily_start = initial_capital
    
    def add_position(self, ticker, shares, entry_price, atr=None, stop_pct=5):
        """添加仓位，自动计算止损"""
        if atr:
            stop_price = entry_price - (atr * 2)
        else:
            stop_price = entry_price * (1 - stop_pct/100)
        
        self.positions[ticker] = {
            'shares': shares,
            'entry': entry_price,
            'stop': stop_price,
            'atr': atr
        }
        print(f"{ticker}: 入场 {entry_price}，止损 {stop_price:.2f}")
    
    def check_risk(self):
        """检查所有仓位的风险"""
        total_risk = 0
        for ticker, pos in self.positions.items():
            position_value = pos['shares'] * pos['entry']
            risk = position_value * (self.max_risk_pct / 100)
            total_risk += risk
        
        print(f"\n当前总风险暴露: {total_risk:.0f}元 ({total_risk/self.capital*100:.2f}%)")
        return total_risk
    
    def should_stop_trading(self):
        """检查是否应该停止交易"""
        daily_loss = self.capital - self.daily_start
        daily_loss_pct = abs(daily_loss / self.daily_start * 100)
        
        if daily_loss_pct >= self.daily_loss_limit_pct:
            print(f"⚠️ 触发日亏损限制 {daily_loss_pct:.2f}%")
            return True
        return False
    
    def update_capital(self, new_capital):
        """每日收盘后更新资金"""
        self.capital = new_capital

# 使用示例
risk_mgr = AutoStopLoss(initial_capital=100000)
risk_mgr.add_position('AAPL', 100, 175, atr=3.5)
risk_mgr.add_position('MSFT', 50, 380, atr=5.2)
risk_mgr.check_risk()
```

### 4.2 动态仓位监控面板

```python
def risk_dashboard(positions, current_prices, capital):
    """
    风险监控面板
    
    返回:
        各仓位当前状态、风险摘要
    """
    print("=" * 60)
    print(f"账户资金: {capital:,.0f}元")
    print("=" * 60)
    
    total_unrealized_pnl = 0
    total_risk = 0
    total_position_value = 0
    
    for ticker, pos in positions.items():
        current = current_prices.get(ticker, pos['entry'])
        entry = pos['entry']
        shares = pos['shares']
        stop = pos.get('stop', entry * 0.95)
        
        # 计算盈亏
        pnl = (current - entry) * shares
        pnl_pct = (current - entry) / entry * 100
        total_unrealized_pnl += pnl
        
        # 计算风险（持仓市值 × 最大风险比例）
        position_value = shares * entry
        position_risk = position_value * 0.02  # 按2%最大风险计算
        total_risk += position_risk
        total_position_value += position_value
        
        # 计算当前风险（距离止损还有多少）
        risk_to_stop = (current - stop) / current * 100
        
        status = "🟢" if pnl > 0 else "🔴"
        print(f"{status} {ticker}: {pnl:+.0f}元 ({pnl_pct:+.1f}%) | 持仓{position_value:.0f}元 | 距止损 {risk_to_stop:.1f}%")
    
    print("-" * 60)
    print(f"未实现盈亏: {total_unrealized_pnl:+.0f}元")
    print(f"总持仓市值: {total_position_value:,.0f}元 ({total_position_value/capital*100:.1f}%)")
    print(f"总风险暴露: {total_risk:,.0f}元 ({total_risk/capital*100:.2f}%)")
    print("=" * 60)

# 示例
positions = {
    'AAPL': {'shares': 100, 'entry': 175, 'stop': 168},
    'MSFT': {'shares': 50, 'entry': 380, 'stop': 365},
}
prices = {'AAPL': 178, 'MSFT': 375}

risk_dashboard(positions, prices, 100000)
```

## ⚠️ 第四部分：常见误区

### 误区1：重仓押注

**错误：** "这个股票肯定涨，全仓进去！"

**正确：** 任何单笔风险不超过总资金 2%。

### 误区2：不止损

**错误：** "亏了没事，早晚涨回来。"

**正确：** 及时止损是风险管理的第一原则。

### 误区3：凯利公式滥用

**错误：** "凯利说可以投 50%，我就投 50%。"

**正确：** 实际使用建议取半凯利或更小。

### 误区4：忽视相关性

**错误：** "我买了5只科技股，很分散。"

**正确：** 如果5只股票高度相关，实际风险集中度很高。

## 📋 速查表

| 方法 | 公式 | 适用场景 |
|------|------|----------|
| 固定仓位 | 资金 × 风险% ÷ 止损% | 简单粗暴，新手入门 |
| ATR动态 | 风险金额 ÷ (ATR×倍数) | 波动市场 |
| 凯利公式 | (bp-q) ÷ b | 长期稳定策略 |

| 规则 | 阈值 | 说明 |
|------|------|------|
| 2%原则 | 单笔≤2% | 任何单笔最大亏损 |
| 日亏损上限 | 3-5% | 触发后停止交易 |
| 凯利仓位 | 半凯利 | 实际使用建议 |

2. **风险管理两原则**：2%单笔限制、日亏损上限
3. **分散化**：关注持仓相关性，不只看数量
4. **自动化**：用代码实现实时风险监控

**记住：** 好的资金管理不能让你一夜暴富，但能让你在市场中活得够久。
