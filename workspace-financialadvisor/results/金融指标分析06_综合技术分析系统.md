## 🌅 开篇：为什么单一指标不够用？

你学会了 5 个技术指标：
- 📊 MA（移动平均线）—— 判断趋势方向
- 📉 RSI（相对强弱指数）—— 判断超买超卖
- 🔀 MACD（指数平滑异同移动平均线）—— 判断动量变化
- 📈 布林带（Bollinger Bands）—— 判断波动率和支撑阻力
- 🎯 ATR（真实波动幅度）—— 设置动态止损

**但问题是：什么时候用哪个？**

想象你要开车去一个陌生的地方。只有地图不够——你需要：
1. 知道目的地在哪里（方向）
2. 知道路线怎么走（入场点）
3. 知道油箱够不够（仓位管理）
4. 知道什么时候该转弯（止损）

**单一指标就像只有地图——你知道大方向，但不知道怎么走完最后一公里。**

> 💡 **类比：** 医生看病需要综合体温、血压、血液指标等多个维度。技术分析也一样——单一指标只能告诉你一个维度的情况。

## 📋 四步交易流程

综合技术分析系统的核心是**四步流程**：

```
第一步：趋势判断 —— 用 MA 判断方向
第二步：入场点选择 —— 用 RSI/MACD/布林带找时机
第三步：止损设置 —— 用 ATR 设置动态止损
第四步：仓位管理 —— 用 ATR 计算仓位
```

### 第一步：趋势判断（用 MA）

**核心原则：顺势而为，不要逆势操作**

```
上涨趋势：价格 > MA20 > MA60
下跌趋势：价格 < MA20 < MA60
震荡趋势：价格围绕 MA20 上下波动
```

**实操判断：**
1. 打开日线图，添加 MA20 和 MA60
2. 观察价格的相对位置
3. 观察 MA 的交叉情况

### 第二步：入场点选择（用 RSI/MACD/布林带）

**核心原则：在趋势方向上的回调结束时入场**

**方案 A：RSI 回调入场**
- 上涨趋势中，RSI 跌到 30-40 然后反弹 = 买入信号
- 下跌趋势中，RSI 涨到 60-70 然后回落 = 卖出信号

**方案 B：MACD 金叉/死叉入场**
- 上涨趋势中，MACD 线从下方穿越信号线 = 买入信号
- 下跌趋势中，MACD 线从上方穿越信号线 = 卖出信号

**方案 C：布林带回调入场**
- 价格触及布林带下轨然后反弹 = 买入信号
- 价格触及布林带上轨然后回落 = 卖出信号

### 第三步：止损设置（用 ATR）

**核心原则：止损位应该给市场"呼吸空间"，但又不能太宽**

```
止损位 = 入场价 - 2 × ATR
```

**为什么用 2×ATR？**
- ATR 反映的是平均日波动幅度
- 2×ATR 给了两天的"呼吸空间"
- 如果跌破这个位置，说明趋势可能已经改变

### 第四步：仓位管理（用 ATR）

**核心原则：每只股票承担相同的风险**

```
仓位 = 账户风险的 1/N / (2 × ATR)

其中：N = 同时持有的股票数量
账户风险 = 账户净值 × 2%
```

**举例：**
- 账户净值 = $50,000
- 单只股票风险 = $50,000 × 2% = $1,000
- 持有 5 只股票 = $1,000 / 5 = $200 风险/只
- 某股票 ATR = $3
- 仓位 = $200 / (2 × $3) = 33 股

## 🖥️ 完整实战案例

### 案例：NVDA 完整交易分析

```python
import yfinance as yf
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

%matplotlib inline
plt.rcParams['font.sans-serif'] = ['Arial Unicode MS', 'SimHei']
plt.rcParams['axes.unicode_minus'] = False

# 获取 NVDA 数据
nvda = yf.Ticker('NVDA')
hist = nvda.history(period='6mo')

# ========== 第一步：趋势判断（MA）==========
hist['MA20'] = hist['Close'].rolling(window=20).mean()
hist['MA60'] = hist['Close'].rolling(window=60).mean()

# 判断趋势
def get_trend(row):
    if row['Close'] > row['MA20'] > row['MA60']:
        return '上涨趋势'
    elif row['Close'] < row['MA20'] < row['MA60']:
        return '下跌趋势'
    else:
        return '震荡'

hist['Trend'] = hist.apply(get_trend, axis=1)

# ========== 第二步：入场点（RSI + MACD + 布林带）==========
# RSI
hist['H-L'] = hist['High'] - hist['Low']
hist['H-PC'] = abs(hist['High'] - hist['Close'].shift(1))
hist['L-PC'] = abs(hist['Low'] - hist['Close'].shift(1))
hist['TR'] = hist[['H-L', 'H-PC', 'L-PC']].max(axis=1)
hist['RSI'] = 100 - (100 / (1 + hist['Close'].pct_change().rolling(14).apply(lambda x: (x[x>0].sum() / -x[x<0].sum()) if x[x<0].sum() != 0 else 100)))

# MACD
hist['EMA12'] = hist['Close'].ewm(span=12, adjust=False).mean()
hist['EMA26'] = hist['Close'].ewm(span=26, adjust=False).mean()
hist['MACD'] = hist['EMA12'] - hist['EMA26']
hist['Signal'] = hist['MACD'].ewm(span=9, adjust=False).mean()
hist['MACD_Hist'] = hist['MACD'] - hist['Signal']

# 布林带
hist['SMA20'] = hist['Close'].rolling(window=20).mean()
hist['STD20'] = hist['Close'].rolling(window=20).std()
hist['UB'] = hist['SMA20'] + 2 * hist['STD20']
hist['LB'] = hist['SMA20'] - 2 * hist['STD20']

# ATR
hist['ATR'] = hist['TR'].ewm(alpha=1/14, min_periods=14).mean()

# ========== 第三步 & 第四步：止损 + 仓位 ==========
# 假设我们今天买入
entry_price = hist['Close'].iloc[-1]
atr = hist['ATR'].iloc[-1]

# 止损
stop_loss = entry_price - 2 * atr

# 仓位（假设账户 $50,000，单只风险 2%，持有 5 只）
account = 50000
risk_per_stock = account * 0.02 / 5  # $200
shares = int(risk_per_stock / (2 * atr))
position_value = shares * entry_price

print("=" * 50)
print("NVDA 完整交易分析")
print("=" * 50)
print(f"\n【第一步：趋势判断】")
print(f"当前趋势: {hist['Trend'].iloc[-1]}")
print(f"收盘价: ${hist['Close'].iloc[-1]:.2f}")
print(f"MA20: ${hist['MA20'].iloc[-1]:.2f}")
print(f"MA60: ${hist['MA60'].iloc[-1]:.2f}")

print(f"\n【第二步：入场点】")
print(f"RSI(14): {hist['RSI'].iloc[-1]:.2f}")
print(f"MACD: {hist['MACD'].iloc[-1]:.2f}")
print(f"Signal: {hist['Signal'].iloc[-1]:.2f}")
print(f"布林带上轨: ${hist['UB'].iloc[-1]:.2f}")
print(f"布林带下轨: ${hist['LB'].iloc[-1]:.2f}")

print(f"\n【第三步：止损设置】")
print(f"入场价: ${entry_price:.2f}")
print(f"ATR(14): ${atr:.2f}")
print(f"止损位 (入场价 - 2×ATR): ${stop_loss:.2f}")
print(f"止损比例: {((stop_loss/entry_price)-1)*100:.2f}%")

print(f"\n【第四步：仓位管理】")
print(f"账户净值: ${account:.2f}")
print(f"单只股票风险: ${risk_per_stock:.2f}")
print(f"建议买入股数: {shares} 股")
print(f"持仓市值: ${position_value:.2f}")
print("=" * 50)
```

**📊 案例结果解读：**
- 趋势判断告诉你现在是上涨还是下跌
- 入场点告诉你什么时候买
- 止损设置告诉你亏多少离场
- 仓位管理告诉你买多少

## 🔍 如何选择指标组合？

### 不同市场环境的指标组合

| 市场环境 | 推荐组合 | 说明 |
|---------|---------|------|
| **趋势明确** | MA + MACD + ATR | MA 确认趋势，MACD 找入场，ATR 止损 |
| **震荡市场** | 布林带 + RSI | 布林带上下轨作为支撑阻力，RSI 找超买超卖 |
| **高波动** | ATR + 布林带 | ATR 设置宽止损，布林带识别极端波动 |
| **低波动** | RSI + MACD | 小幅波动中找动量变化 |

### 四步流程的选择逻辑

```
1. 先用 MA 判断趋势方向
   ↓
2. 如果趋势明确（上涨/下跌）：
   - 用 RSI 或 MACD 找回调入场点
   ↓
3. 如果趋势不明确（震荡）：
   - 用布林带找区间上下沿
   ↓
4. 最后用 ATR 设置止损和仓位
```

## ⚠️ 常见错误和避坑指南

### ❌ 错误 1：逆势操作

**错误做法：** 下跌趋势中 RSI 超卖就买

**正确做法：** 下跌趋势中 RSI 超卖只是反弹信号，要等趋势反转后再买

### ❌ 错误 2：止损设置太紧

**错误做法：** 止损设置在 1×ATR，市场正常波动就被洗出

**正确做法：** 止损至少 2×ATR，给市场呼吸空间

### ❌ 错误 3：仓位太重

**错误做法：** 看好一只股票就 All In

**正确做法：** 用 ATR 仓位管理，每只股票承担相同风险

### ❌ 错误 4：不止损

**错误做法：** 亏了就死扛，等回本

**正确做法：** 严格按照 ATR 止损，亏到止损位就走

### ❌ 错误 5：同时用太多指标

**错误做法：** MA、RSI、MACD、布林带、ATR 同时看，信号冲突时不知道怎么办

**正确做法：** 每个步骤只用 1-2 个指标，信号冲突时以上一级为准

## 🧠 思考题

### 思考题 1 ✅ 含参考答案

**题目：**
某股票：
- 上涨趋势（价格 > MA20 > MA60）
- RSI = 35（回调到超卖区域）
- ATR = $5
- 入场价 = $100

请给出完整的交易计划（入场价、止损价、仓位）。

**参考答案：**
1. 趋势判断：上涨趋势 → 可以买入
2. 入场点：RSI = 35 在超卖区域，可以买入
3. 止损 = 入场价 - (2 × ATR)
           = $100 - (2 × $5)
           = $100 - $10
           = $90（下跌 10%）
4. 假设账户 $50,000，单只风险 $200（0.4%）
5. 仓位 = 风险金额 / (2 × ATR)
             = $200 / (2 × $5)
             = $200 / $10
             = 20 股，持仓市值 $2,000

交易计划：
- 买入价：$100
- 止损价：$90
- 买入股数：20 股
- 持仓市值：$2,000
- 风险金额：$200（2% × $50,000 / 5 只）

### 思考题 2 ✅ 含参考答案

**题目：**
如果 MACD 和 RSI 同时发出买入信号，但 MA 显示是下跌趋势，应该怎么做？

**参考答案：**
1. **以下跌趋势为准**，不要买入
2. 原因：逆势操作的胜率很低
3. 正确做法：等待 MA 显示趋势反转（价格站上 MA20 > MA60）后，再用 RSI/MACD 找入场点
4. 或者选择其他处于上涨趋势的股票

---

### 思考题 3 ✅ 含参考答案

**题目：**
布林带收口（挤压）后，价格向上突破上轨，但 MA 显示是下跌趋势。RSI 显示超买。这时候应该买吗？

**参考答案：**
1. **不应该买**
2. 原因：
   - 布林带突破可能是假突破（被压迫后的正常反弹）
   - 下跌趋势中，任何反弹都是卖出机会
   - RSI 超买说明短期内可能回调
3. 正确做法：等待下跌趋势结束（价格站上 MA20 > MA60），然后等回调到布林带中轨或下轨时再考虑买入

### 练习 2（进阶）：模拟回测 ✅ 含验收标准

用历史数据模拟一个简单的基于 MA 的交易策略，计算收益率和最大回撤。

**验收标准：** 运行代码后，你应该能看到：
1. 策略收益率 vs 买入持有收益率
2. 最大回撤
3. 交易次数统计
4. 胜率统计

**📋 练习2 验收标准：**
1. ✅ 策略收益率 vs 买入持有收益率
2. ✅ 最大回撤
3. ✅ 交易次数统计
4. ✅ 胜率统计

**策略逻辑：** MA20 上穿 MA60 买入，下穿 MA60 卖出。

```python
import yfinance as yf
import pandas as pd
import numpy as np

ticker = 'SPY'  # 换成你想回测的ETF或股票

stock = yf.Ticker(ticker)
hist = stock.history(period='2y')

# 计算 MA
hist['MA20'] = hist['Close'].rolling(window=20).mean()
hist['MA50'] = hist['Close'].rolling(window=50).mean()

# 生成交易信号
hist['Signal'] = 0  # 0 = 空仓, 1 = 持仓
hist.loc[hist['MA20'] > hist['MA60'], 'Signal'] = 1  # 金叉买入
hist.loc[hist['MA20'] <= hist['MA60'], 'Signal'] = 0  # 死叉卖出

# 计算每日收益率
hist['Returns'] = hist['Close'].pct_change()

# 策略收益（持仓时才有收益）
hist['Strategy_Returns'] = hist['Returns'] * hist['Signal'].shift(1)

# 计算累计收益
hist['Cumulative_Returns'] = (1 + hist['Returns']).cumprod()
hist['Cumulative_Strategy'] = (1 + hist['Strategy_Returns']).cumprod()

# 最大回撤
cumulative = hist['Cumulative_Strategy']
running_max = cumulative.expanding().max()
drawdown = (cumulative - running_max) / running_max
max_drawdown = drawdown.min()

# 交易统计
trades = []
in_position = False
for i in range(len(hist)):
    if hist['Signal'].iloc[i] == 1 and not in_position:
        # 买入
        trades.append({'type': 'buy', 'price': hist['Close'].iloc[i], 'date': hist.index[i]})
        in_position = True
    elif hist['Signal'].iloc[i] == 0 and in_position:
        # 卖出
        trades.append({'type': 'sell', 'price': hist['Close'].iloc[i], 'date': hist.index[i]})
        in_position = False

# 计算胜率
winning_trades = 0
total_trades = len([t for t in trades if t['type'] == 'sell'])
for i in range(0, len(trades)-1, 2):
    if i+1 < len(trades):
        if trades[i+1]['price'] > trades[i]['price']:
            winning_trades += 1

win_rate = winning_trades / total_trades if total_trades > 0 else 0

# 打印结果
print("=" * 50)
print(f"{ticker} MA 交叉策略回测")
print("=" * 50)
print(f"\n【收益对比】")
print(f"买入持有收益率: {(hist['Cumulative_Returns'].iloc[-1]-1)*100:+.2f}%")
print(f"MA 策略收益率: {(hist['Cumulative_Strategy'].iloc[-1]-1)*100:+.2f}%")

print(f"\n【风险统计】")
print(f"最大回撤: {max_drawdown*100:.2f}%")

print(f"\n【交易统计】")
print(f"总交易次数: {total_trades}")
print(f"胜率: {win_rate*100:.2f}%")

if trades:
    print(f"\n【最近一笔交易】")
    last_trade = trades[-1]
    print(f"类型: {last_trade['type'].upper()}")
    print(f"价格: ${last_trade['price']:.2f}")
    print(f"日期: {last_trade['date'].date()}")

print("=" * 50)
print("\n✅ 验收标准 1-4 全部通过则练习完成！")
```

### 练习 3（挑战）：多指标综合策略

用 RSI、MACD、布林带三个指标综合判断，设计一个日内交易策略，并回测验证。

**验收标准：** 运行代码后，你应该能看到：
1. 策略逻辑说明
2. 回测收益率
3. 最大回撤
4. 每日交易信号统计

```python
# ===== 验收标准检查 =====
# 运行后检查以下几点：
# 1. ✅ 策略逻辑说明
# 2. ✅ 回测收益率
# 3. ✅ 最大回撤
# 4. ✅ 每日交易信号统计
# =========================

import yfinance as yf
import pandas as pd
import numpy as np

ticker = 'QQQ'  # 换成你想回测的ETF

stock = yf.Ticker(ticker)
hist = stock.history(period='1y')

# 计算指标
# RSI
delta = hist['Close'].diff()
gain = delta.where(delta > 0, 0).rolling(14).mean()
loss = (-delta.where(delta < 0, 0)).rolling(14).mean()
rs = gain / loss
hist['RSI'] = 100 - (100 / (1 + rs))

# MACD
hist['EMA12'] = hist['Close'].ewm(span=12, adjust=False).mean()
hist['EMA26'] = hist['Close'].ewm(span=26, adjust=False).mean()
hist['MACD'] = hist['EMA12'] - hist['EMA26']
hist['Signal'] = hist['MACD'].ewm(span=9, adjust=False).mean()

# 布林带
hist['SMA20'] = hist['Close'].rolling(window=20).mean()
hist['STD20'] = hist['Close'].rolling(window=20).std()
hist['UB'] = hist['SMA20'] + 2 * hist['STD20']
hist['LB'] = hist['SMA20'] - 2 * hist['STD20']

# 综合交易信号
# 买入信号：RSI < 30 且 MACD 金叉 且价格触及布林带下轨
# 卖出信号：RSI > 70 或 价格触及布林带上轨
hist['Buy_Signal'] = (hist['RSI'] < 30) & (hist['MACD'] > hist['Signal']) & (hist['Close'] <= hist['LB'] * 1.02)
hist['Sell_Signal'] = (hist['RSI'] > 70) | (hist['Close'] >= hist['UB'] * 0.98)

# 回测
hist['Position'] = 0
position = 0
for i in range(len(hist)):
    if hist['Buy_Signal'].iloc[i] and position == 0:
        position = 1
    elif hist['Sell_Signal'].iloc[i] and position == 1:
        position = 0
    hist.loc[hist.index[i], 'Position'] = position

hist['Returns'] = hist['Close'].pct_change()
hist['Strategy_Returns'] = hist['Returns'] * hist['Position'].shift(1)

# 计算收益（扣除交易成本）
transaction_cost = 0.001  # 假设每笔交易成本 0.1%
hist['Strategy_Returns'] = hist['Strategy_Returns'] - transaction_cost

hist['Cumulative'] = (1 + hist['Strategy_Returns']).cumprod()

# 最大回撤
cumulative = hist['Cumulative']
running_max = cumulative.expanding().max()
drawdown = (cumulative - running_max) / running_max
max_drawdown = drawdown.min()

# 统计
total_days = len(hist)
buy_signals = hist['Buy_Signal'].sum()
sell_signals = hist['Sell_Signal'].sum()

print("=" * 50)
print(f"{ticker} 多指标综合策略回测")
print("=" * 50)
print(f"\n【策略逻辑】")
print("买入信号：RSI < 30 且 MACD 金叉 且价格触及布林带下轨")
print("卖出信号：RSI > 70 或 价格触及布林带上轨")

print(f"\n【收益统计】")
print(f"策略收益率: {(hist['Cumulative'].iloc[-1]-1)*100:+.2f}%")

print(f"\n【风险统计】")
print(f"最大回撤: {max_drawdown*100:.2f}%")

print(f"\n【信号统计】")
print(f"回测天数: {total_days}")
print(f"买入信号次数: {buy_signals}")
print(f"卖出信号次数: {sell_signals}")
print(f"日均信号: {buy_signals/total_days*100:.2f}%")

print("=" * 50)
print("\n✅ 验收标准 1-4 全部通过则练习完成！")
```

### 🎯 核心知识点

| 概念 | 关键点 |
|------|--------|
| **四步流程** | 趋势判断 → 入场点 → 止损设置 → 仓位管理 |
| **趋势判断** | 用 MA20 + MA60，确认方向后再操作 |
| **入场点** | RSI/MACD/布林带，顺势回调时入场 |
| **止损** | ATR 动态止损，给市场呼吸空间 |
| **仓位** | ATR 等风险持仓，每只股票承担相同风险 |
| **避坑** | 不要逆势、止损别太紧、仓位别太重 |
