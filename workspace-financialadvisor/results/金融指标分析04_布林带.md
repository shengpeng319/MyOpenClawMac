## 🌅 开篇：价格的"体温计"

想象你在观察股票的"体温"。

股票市场和人一样，也有"体温"——有时活跃（高波动），有时平静（低波动）。布林带就是一种**波动性体温计**，用它你可以：
- 看到价格的"正常波动范围"在哪里
- 判断当前价格是否已经"发烧"或"体温过低"
- 预警即将到来的大幅波动

> 💡 **生活中的类比：** 想象你在量血压。收缩压和舒张压（上下轨）定义了一个"正常范围"。如果你的血压经常超出这个范围，不管往上还是往下，都需要引起注意。布林带的作用类似——告诉你在什么情况下价格已经"偏高"或"偏低"。但要注意：血压超标不意味着立刻要反向操作，而是提醒你"处于极端状态，需要关注"。

## 📊 什么是布林带？

布林带由三条线组成：

| 组成部分 | 计算方法 | 作用 |
|---------|---------|------|
| **中轨（MB）** | 20日简单移动平均线（SMA） | 定义中期趋势方向 |
| **上轨（UB）** | 中轨 + 2倍标准差 | 定义"正常范围"的上限 |
| **下轨（LB）** | 中轨 - 2倍标准差 | 定义"正常范围"的下限 |

### 为什么是 ±2 标准差？

**统计学原理：**

在正态分布假设下，±2σ 覆盖约 95% 的数据。这是一个**数学性质**，不是布林带"选择"了 95%。

| 标准差范围 | 数据覆盖（正态分布假设）|
|-----------|---------------------|
| ±1σ | 约 68% |
| ±2σ | 约 95% |
| ±3σ | 约 99.7% |

> ⚠️ **重要前提：** 金融市场的价格分布**并非正态分布**——极端事件（"肥尾"）比正态分布预测的更多。所以布林带是**风险可视化工具**，不是精确预测工具。触碰上轨不一定马上跌，可能继续涨；触碰下轨不一定马上涨，可能继续跌。

> ⚠️ **前置知识：** 标准差是衡量数据离散程度的指标。如果你对标准差不熟悉，可以这样理解：标准差越大，数据越分散；标准差越小，数据越集中。举例：A股每天涨跌1% vs 美股每天涨跌2%，美股的标准差更大。

**手算示例：**
假设5天收盘价：$100, $102, $101, $103, $99

```
第1步：计算平均值 = (100+102+101+103+99)/5 = $101
第2步：计算每个偏差 = 收盘价 - 平均值
  = -1, +1, 0, +2, -2
第3步：平方偏差 = 1, 1, 0, 4, 4
第4步：计算方差 = (1+1+0+4+4)/5 = 2
第5步：标准差 = √2 ≈ $1.41
```

## 🔢 计算方法

### 布林带三步计算

**第一步：计算 20 日简单移动平均线（SMA）**
```
中轨 = 最近20天收盘价的简单平均
```

**第二步：计算 20 天标准差**
```
标准差 = √[Σ(收盘价 - 平均价)² / N]
```

**第三步：计算上轨和下轨**
```
上轨 = 中轨 + 2 × 标准差
下轨 = 中轨 - 2 × 标准差
```

### Python 代码

```python
import yfinance as yf
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

# 获取数据
stock = yf.Ticker('AAPL')
hist = stock.history(period='6mo')

# 计算布林带
hist['SMA20'] = hist['Close'].rolling(window=20).mean()
hist['STD20'] = hist['Close'].rolling(window=20).std()
hist['UB'] = hist['SMA20'] + 2 * hist['STD20']  # 上轨
hist['LB'] = hist['SMA20'] - 2 * hist['STD20']  # 下轨

# 查看结果
print(hist[['Close', 'SMA20', 'UB', 'LB']].tail(10))
```

## 📈 布林带的三大形态

### 1️⃣ 开口（Expansion）—— 波动加剧

**现象：** 上轨和下轨之间的距离变大

**文字描述：**
```
上轨:    ↗━━━━━━━↗
          ↗ 开口  ↗
中轨:    →━━━━━━━→
          ↘ 开口  ↘
下轨:    ↘━━━━━━━↘
```

**为什么重要？**
- 开口通常发生在**趋势启动或加速阶段**
- 波动性增加，市场情绪激动
- 顺势交易的机会

---

### 2️⃣ 收口（Contraction）—— 波动减缓

**现象：** 上轨和下轨之间的距离变小

**文字描述：**
```
上轨:    ↘━━━━━↘
          ↘收口↘
中轨:    →━━━━━→
          ↗收口↗
下轨:    ↗━━━━━↗
```

**为什么重要？**
- 收口通常发生在**趋势结束或横盘阶段**
- 波动性降低，市场在"休息"
- 可能是趋势反转或新趋势启动的前兆

---

### 3️⃣ 挤压（Squeeze）—— 蓄势待发

**挤压是布林带最独特的信号！**

**现象：** 布林带收口到极端程度（上下轨之间的距离达到历史最低水平）

**为什么叫"挤压"？**
- 像弹簧被压缩到极限
- 能量在积累，一旦释放，波动会很大
- **挤压不是买卖信号，但它是重要预警**

**挤压之后的两种走法：**
| 情况 | 后续走势 | 布林带形态 |
|------|---------|-----------|
| 向上突破 | 上涨 | 上轨向上开口 + 价格突破上轨 |
| 向下突破 | 下跌 | 下轨向下开口 + 价格跌破下轨 |

**如何识别挤压？**
```python
# 计算布林带宽度（标准化指标）
hist['BB_width'] = (hist['UB'] - hist['LB']) / hist['SMA20']

# 布林带宽度创20天新低 = 挤压信号
hist['BB_squeeze'] = hist['BB_width'] < hist['BB_width'].rolling(20).min()

print("挤压信号日期：")
print(hist[hist['BB_squeeze']].index)
```

## ⚠️ 布林带的局限性

### ❌ 不能独立预测趋势

布林带只能告诉你价格是否在"极端"位置，不能告诉你趋势往哪个方向走。

**✅ 正确做法：** 结合 MACD（第五课内容）判断趋势方向，用布林带找入场点。

---

### ❌ 假突破频繁

价格经常触碰上轨或下轨，然后迅速回到布林带内——这就是**假突破**。

**✅ 正确做法：** 结合 RSI（第三课内容）过滤假突破（触碰上轨 + RSI 超买 = 可能是假突破）。

---

### ❌ 不适合横盘震荡

在横盘震荡行情中，布林带会反复"挤压-开口-假突破"，导致频繁亏损。

**✅ 正确做法：** 布林带更适合有明显趋势的市场。在震荡市中谨慎使用。

**📊 案例结果解读：**

- 价格 < 下轨：可能超卖
- 开口和收口反映波动性变化

### 案例 2：布林带 + RSI 组合——识别假突破

**为什么这个组合有效？**
- 布林带告诉你价格是否在"极端"位置
- RSI 告诉你是否真的超买超卖
- 两者结合 = 布林带定位 + RSI 确认

**策略逻辑：**
- 买入信号：价格跌破下轨 + RSI < 30（双重超卖）
- 卖出信号：价格突破上轨 + RSI > 70（双重超买）

```python
# 计算 RSI（Wilder's 平滑法）
def calculate_rsi(prices, period=14):
    delta = prices.diff()
    gain = delta.where(delta > 0, 0)
    loss = -delta.where(delta < 0, 0)
    avg_gain = gain.ewm(alpha=1/period, min_periods=period).mean()
    avg_loss = loss.ewm(alpha=1/period, min_periods=period).mean()
    rs = avg_gain / avg_loss
    rs = rs.replace([np.inf, -np.inf], 0)
    rsi = 100 - (100 / (1 + rs))
    return rsi

hist['RSI'] = calculate_rsi(hist['Close'])

# 计算布林带
hist['SMA20'] = hist['Close'].rolling(window=20).mean()
hist['STD20'] = hist['Close'].rolling(window=20).std()
hist['UB'] = hist['SMA20'] + 2 * hist['STD20']
hist['LB'] = hist['SMA20'] - 2 * hist['STD20']

# 布林带 + RSI 买入信号（价格跌破下轨，且RSI确认超卖）
hist['BuySignal'] = (hist['Close'] < hist['LB']) & (hist['RSI'] < 30)

# 布林带 + RSI 卖出信号（价格突破上轨，且RSI确认超买）
hist['SellSignal'] = (hist['Close'] > hist['UB']) & (hist['RSI'] > 70)

# 查看信号
buy_dates = hist[hist['BuySignal']].index
sell_dates = hist[hist['SellSignal']].index

print(f"布林带+RSI 买入信号数量: {len(buy_dates)}")
print(f"布林带+RSI 卖出信号数量: {len(sell_dates)}")

# 打印最近的买入信号
if len(buy_dates) > 0:
    print("\n=== 最近买入信号 ===")
    print(hist.loc[buy_dates[-5:], ['Close', 'RSI', 'UB', 'LB']])
```

**📊 案例结果解读：**
- 双重确认（布林带 + RSI）可以过滤假突破
- 注意：价格跌破下轨用 `<`（严格小于），不是 `<=`
- 信号少但精准，适合长线投资者

### 案例 3：布林带 + MACD 组合——趋势确认

**为什么这个组合有效？**
- MACD 确认趋势方向（第五课内容）
- 布林带确认入场时机
- 顺势而为，提高胜率

**策略逻辑：**
- 只做 MACD 多头的股票（DIF > DEA 且都在零轴上方）
- 在布林带下轨附近买入（顺势抄底）
- 止损用固定百分比，不用下轨（因为下轨距离可能很大）

```python
# 计算 MACD
hist['EMA12'] = hist['Close'].ewm(span=12, adjust=False).mean()
hist['EMA26'] = hist['Close'].ewm(span=26, adjust=False).mean()
hist['DIF'] = hist['EMA12'] - hist['EMA26']
hist['DEA'] = hist['DIF'].ewm(span=9, adjust=False).mean()

# 计算布林带
hist['SMA20'] = hist['Close'].rolling(window=20).mean()
hist['STD20'] = hist['Close'].rolling(window=20).std()
hist['UB'] = hist['SMA20'] + 2 * hist['STD20']
hist['LB'] = hist['SMA20'] - 2 * hist['STD20']

# MACD 多头 + 布林带下轨支撑买入
hist['BuySignal'] = (hist['DIF'] > hist['DEA']) & (hist['DEA'] > 0) & (hist['Close'] < hist['LB'])

# 止损：固定百分比（3%），不用下轨下方（距离可能太大）
hist['StopLoss'] = hist['Close'] * 0.97

# 查看信号
buy_dates = hist[hist['BuySignal']].index
print(f"\nMACD+布林带 买入信号数量: {len(buy_dates)}")
if len(buy_dates) > 0:
    print("\n=== 最近买入信号 ===")
    print(hist.loc[buy_dates[-5:], ['Close', 'RSI', 'UB', 'LB', 'DIF', 'DEA', 'StopLoss']])
```

**📊 案例结果解读：**
- 止损用固定百分比（3%）而不是"下轨下方"
- 如果用下轨下方止损，距离可能达到 2 个标准差，风险太大
- 顺势操作才能提高胜率

### 案例 4：识别挤压信号（Squeeze）

```python
# 计算布林带宽度
hist['BB_width'] = (hist['UB'] - hist['LB']) / hist['SMA20']

# 挤压信号：布林带宽度创20天新低
hist['BB_squeeze'] = hist['BB_width'] < hist['BB_width'].rolling(20).min()

# 查找挤压信号后的突破
squeeze_dates = hist[hist['BB_squeeze']].index
print(f"挤压信号次数: {len(squeeze_dates)}")

# 挤压之后30天内是否有效突破？
突破统计 = []
for sd in squeeze_dates:
    post_data = hist[hist.index > sd].head(30)
    if len(post_data) > 5:
        max_price = post_data['Close'].max()
        min_price = post_data['Close'].min()
        last_price = hist.loc[sd, 'Close']
        upper突破 = max_price > hist.loc[sd, 'UB']
        lower突破 = min_price < hist.loc[sd, 'LB']
        突破统计.append({
            '日期': sd.date(),
            '突破方向': '向上' if upper突破 else ('向下' if lower突破 else '无'),
            '幅度': f"{(max_price/last_price - 1)*100:.1f}%" if upper突破 else f"{(1 - min_price/last_price)*100:.1f}%"
        })

print("\n=== 挤压后突破统计 ===")
for t in 突破统计[-10:]:
    print(f"{t['日期']}: {t['突破方向']} {t['幅度']}")
```

## ⚠️ 常见误区

### ❌ 误区 1：价格触碰上轨就卖，下轨就买

**为什么不对？**
- 趋势强劲时，价格可以沿着布林带上轨/下轨运行很久
- 逆势抄底可能亏损惨重

**✅ 正确做法：**
- 只在 MACD 多头时做多，MACD 空头时做空
- 用布林带找具体入场点，不是判断方向

---

### ❌ 误区 2：挤压一定会有大行情

**为什么不对？**
- 挤压只是说明波动性降低，不保证会有大行情
- 有时候挤压之后继续保持横盘

**✅ 正确做法：**
- 挤压是预警信号，不是确认信号
- 等有效突破（上轨或下轨）再行动

---

### ❌ 误区 3：布林带参数不需要调整

**为什么不对？**
- 默认参数 20/2（20日均线、±2标准差）是通用设置
- 高波动股票可能需要更宽的带（±3标准差）
- 低波动股票可能用 ±1.5 标准差更合适

**✅ 正确做法：**
| 股票类型 | 建议布林带参数 |
|---------|--------------|
| 高波动成长股 | 20日 / ±3σ |
| 普通股票 | 20日 / ±2σ（默认）|
| 低波动蓝筹股 | 20日 / ±1.5σ |

---

### ❌ 误区 4：布林带可以单独使用

**为什么不对？**
- 布林带是**滞后指标**（依赖历史数据）
- 布林带不提供趋势方向信息

**✅ 正确做法：**
- 始终结合 MACD 或均线判断趋势方向
- 只在顺势方向使用布林带找入场点

## 📋 布林带信号速查表

| 信号类型 | 判断条件 | 信号强度 | 操作建议 |
|---------|---------|---------|---------|
| **下轨支撑买入** | 价格跌破下轨 + RSI < 30 | ⭐⭐⭐⭐ 强 | 买入（顺势）|
| **上轨压力卖出** | 价格突破上轨 + RSI > 70 | ⭐⭐⭐⭐ 强 | 卖出（顺势）|
| **挤压预警** | 布林带宽度创历史新低 | ⭐⭐⭐ 中 | 等待突破，确认方向 |
| **向上突破** | 价格突破上轨 + 开口 + 放量 | ⭐⭐⭐⭐⭐ 极强 | 顺势买入/持有 |
| **向下突破** | 价格跌破下轨 + 开口 + 放量 | ⭐⭐⭐⭐⭐ 极强 | 顺势卖出/做空 |

> ⚠️ **重要区分：**
> - **价格触碰上轨 ≠ 向上突破**：触碰可能是假突破，需要放量+开口确认
> - **MACD 多头 + 价格触碰上轨 ≠ 买入信号**：说明价格已经过热，不应追高

### 思考题 1 ✅ 含参考答案

**题目：**
某股票当前状态：
- 收盘价 = $105
- 中轨 = $100
- 上轨 = $110
- 下轨 = $90
- RSI = 68

请判断：
1. 当前价格位置在布林带的哪里？
2. 这个位置 + RSI = 68 意味着什么？

**参考答案方向：**
1. 答案：价格在**上轨与中轨之间**（$105 在 $110 和 $100 之间）
2. 答案：价格偏高但未超买（RSI 68 < 70），可能接近超买区域，但不是强烈卖出信号。需要观察是否继续上涨突破上轨。

### 思考题 2 ✅ 含参考答案

**题目：**
布林带"挤压"信号出现后，价格一定会突破吗？结合案例4的统计数据，谈谈你的理解。

2. 结合案例4的统计数据：挤压后可能有向上突破、向下突破、或者无有效突破三种情况。

### 思考题 3 ✅ 含参考答案

**题目：**
"MACD 多头 + 价格触碰布林带上轨" 这种情况下应该买入吗？为什么？

**参考答案方向：**
1. 答案：**不应该买入**，反而应该警惕过热风险。
2. 原因：
   - MACD 多头 = 上升趋势确认
   - 价格触碰上轨 = 可能已经进入极端区域
   - 两者结合 = **趋势强劲但价格可能过热**
3. 区分"触碰"和"突破"：
   - 触碰上轨 ≠ 向上突破（突破需要放量+开口确认）
   - MACD 多头时触碰上轨，可能是强势，但不建议追高
4. 正确做法：持有仓位可以继续持有，等待价格回落至中轨附近再考虑加仓。

### 练习 2（进阶）：布林带 + RSI 组合 ✅ 含验收标准

修改练习1的代码，添加 RSI 指标，并用布林带 + RSI 组合找出买入和卖出信号。

**验收标准：** 运行代码后，你应该能看到：
1. 上半部分：股价 + 布林带
2. 下半部分：RSI 曲线（标注 30 和 70 线）
3. 买入信号标注（价格跌破下轨 + RSI < 30）
4. 卖出信号标注（价格突破上轨 + RSI > 70）
5. 打印出所有买入和卖出信号的日期

```python
# ===== 验收标准检查 =====
# 运行后检查以下几点：
# 1. ✅ 上半部分：股价 + 布林带（中轨/上轨/下轨）
# 2. ✅ 下半部分：RSI 曲线（标注 30 和 70 线）
# 3. ✅ 买入信号标注（价格跌破下轨 + RSI < 30）
# 4. ✅ 卖出信号标注（价格突破上轨 + RSI > 70）
# 5. ✅ 打印出所有买入和卖出信号的日期
# =========================

# 计算 RSI
def calculate_rsi(prices, period=14):
    delta = prices.diff()
    gain = delta.where(delta > 0, 0)
    loss = -delta.where(delta < 0, 0)
    avg_gain = gain.ewm(alpha=1/period, min_periods=period).mean()
    avg_loss = loss.ewm(alpha=1/period, min_periods=period).mean()
    rs = avg_gain / avg_loss
    rs = rs.replace([np.inf, -np.inf], 0)
    rsi = 100 - (100 / (1 + rs))
    return rsi

hist['RSI'] = calculate_rsi(hist['Close'])

# 计算布林带
hist['SMA20'] = hist['Close'].rolling(window=20).mean()
hist['STD20'] = hist['Close'].rolling(window=20).std()
hist['UB'] = hist['SMA20'] + 2 * hist['STD20']
hist['LB'] = hist['SMA20'] - 2 * hist['STD20']

# 布林带 + RSI 信号（注意：用 < 和 > 严格判断，不用 <= 或 >=）
hist['BuySignal'] = (hist['Close'] < hist['LB']) & (hist['RSI'] < 30)
hist['SellSignal'] = (hist['Close'] > hist['UB']) & (hist['RSI'] > 70)

# 绘制
fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(14, 10), sharex=True)
# 上半部分：股价 + 布林带
ax1.plot(hist.index, hist['Close'], label=f'{ticker} 收盘价', color='black', linewidth=1.5)
ax1.plot(hist.index, hist['SMA20'], label='中轨', color='blue', linewidth=1)
ax1.plot(hist.index, hist['UB'], label='上轨', color='red', linewidth=1, linestyle='--')
ax1.plot(hist.index, hist['LB'], label='下轨', color='green', linewidth=1, linestyle='--')
ax1.scatter(hist.loc[hist['BuySignal']].index, hist.loc[hist['BuySignal'], 'Close'], 
            marker='^', color='green', s=100, label='买入信号')
ax1.scatter(hist.loc[hist['SellSignal']].index, hist.loc[hist['SellSignal'], 'Close'], 
            marker='v', color='red', s=100, label='卖出信号')
ax1.set_title(f'{ticker} 布林带 + RSI 分析')
ax1.legend()
ax1.grid(True, alpha=0.3)

# 下半部分：RSI
ax2.plot(hist.index, hist['RSI'], label='RSI', color='purple')
ax2.axhline(y=30, color='green', linestyle='--', label='超卖线 (30)')
ax2.axhline(y=70, color='red', linestyle='--', label='超买线 (70)')
ax2.set_ylabel('RSI')
ax2.legend()
ax2.grid(True, alpha=0.3)
ax2.set_ylim(0, 100)

plt.tight_layout()
plt.show()

# 打印验收结果
buy_dates = hist[hist['BuySignal']].index
sell_dates = hist[hist['SellSignal']].index
print(f"\n=== 验收结果 ===")
print(f"买入信号数量: {len(buy_dates)}")
print(f"卖出信号数量: {len(sell_dates)}")
if len(buy_dates) > 0:
    print(f"最近买入信号: {buy_dates[-1].date()}")
if len(sell_dates) > 0:
    print(f"最近卖出信号: {sell_dates[-1].date()}")
print("\n✅ 验收标准 1-5 全部通过则练习完成！")
```

### 练习 3（挑战）：识别挤压信号

用案例4的方法，检测任意股票的挤压信号，并统计挤压之后的突破方向和幅度。

**验收标准：** 运行代码后，你应该能看到：
1. 所有挤压信号的日期
2. 每个挤压信号发生后30天内的突破方向（向上/向下/无）
3. 每个突破的幅度百分比

```python
# 计算布林带
hist['SMA20'] = hist['Close'].rolling(window=20).mean()
hist['STD20'] = hist['Close'].rolling(window=20).std()
hist['UB'] = hist['SMA20'] + 2 * hist['STD20']
hist['LB'] = hist['SMA20'] - 2 * hist['STD20']

# 计算布林带宽度
hist['BB_width'] = (hist['UB'] - hist['LB']) / hist['SMA20']

# 挤压信号：布林带宽度创20天新低
hist['BB_squeeze'] = hist['BB_width'] < hist['BB_width'].rolling(20).min()

# 查找挤压信号后的突破
squeeze_dates = hist[hist['BB_squeeze']].index
print(f"=== 挤压信号统计 ===")
print(f"挤压信号总次数: {len(squeeze_dates)}")

突破统计 = []
for sd in squeeze_dates:
    post_data = hist[hist.index > sd].head(30)
    if len(post_data) > 5:
        max_price = post_data['Close'].max()
        min_price = post_data['Close'].min()
        last_price = hist.loc[sd, 'Close']
        upper突破 = max_price > hist.loc[sd, 'UB']
        lower突破 = min_price < hist.loc[sd, 'LB']
        突破统计.append({
            '日期': sd.date(),
            '突破方向': '向上' if upper突破 else ('向下' if lower突破 else '无'),
            '幅度': f"{(max_price/last_price - 1)*100:.1f}%" if upper突破 else (f"{(1 - min_price/last_price)*100:.1f}%" if lower突破 else '-')
        })

print("\n=== 挤压后突破统计 ===")
for t in 突破统计:
    print(f"{t['日期']}: {t['突破方向']} {t['幅度']}")
```

## 📋 本课小结

### 🎯 核心知识点

| 概念 | 关键点 |
|------|--------|
| **布林带组成** | 中轨（SMA20）+ 上轨（+2σ）+ 下轨（-2σ）|
| **±2σ 原因** | 正态分布数学性质，覆盖95%数据（金融有肥尾）|
| **标准差计算** | √[Σ(偏差²)/N]，手算示例见正文 |
| **开口** | 上下轨距离扩大 = 波动加剧 = 趋势加速 |
| **收口** | 上下轨距离缩小 = 波动降低 = 趋势减缓 |
| **挤压** | 布林带宽度创历史新低 = 蓄势待发 = 重要预警 |
| **假突破** | 价格触碰轨道后迅速回到带内，需 RSI 确认 |
| **布林带 + RSI** | 双重确认超买超卖，过滤假信号 |
| **布林带 + MACD** | 趋势确认 + 入场点选择 |
| **触碰 ≠ 突破** | 触碰上轨需要放量+开口才是真突破 |

### 💡 记住这句话

> **"布林带不是预测工具，而是风险可视化工具。挤压是蓄势，突破是释放。触碰上轨不等于向上突破——MACD多头时碰触上轨是过热信号，不应追高。"**
