# OpenClaw 金融投资辅助最佳实践

*Created: 2026-03-21*

---

## 核心理念

OpenClaw 做"眼"和"嘴"，Claude Code 做"手"。

| 助手 | 定位 | 适合场景 |
|------|------|---------|
| **OpenClaw** | 投资秘书（7x24待命） | 持续监控、日常管理、消息触达 |
| **Claude Code** | 量化工程师 | 构建工具、复杂分析、批量处理 |

---

## OpenClaw 投资功能矩阵

### 1. 每日定时任务

| 时间 | 内容 | 目的 |
|------|------|------|
| 周一 9:00 | 上周市场回顾 + 本周关注事件 | 一周布局参考 |
| 周二～四 8:30 | 美股盘前快讯 | 开盘前热身 |
| 周五 18:00 | 本周总结 + 下周重要财报/事件日历 | 做好准备 |
| 财报季 | 盘后及时分析重点公司财报 | 抓住机会 |

### 2. 事件驱动告警

| 触发条件 | 动作 |
|---------|------|
| 股价跌破阈值 | 飞书告警 |
| 持仓股重大利空 | 立即推送 |
| 美联储利率决议 | 即时分析 |
| 地缘冲突爆发 | 板块影响评估 |

### 3. 研究辅助

- 搜索新闻、分析研报
- 对比多只股票基本面
- 筛选股票（按财务指标）
- 风险提示

---

## Claude Code 配合场景

当需要以下任务时，调用 Claude Code：
- 构建量化交易脚本
- 批量处理历史数据做回测
- 搭建自动化交易机器人
- 复杂财务模型分析
- 用 Python 处理大规模数据集
- 搭建金融数据库

调用方式：
```bash
# 在 OpenClaw 中触发
spawn Claude Code / OpenCode
```

## 风险提示

⚠️ **OpenClaw 不能预测股价**，只能辅助：
- 收集和分析公开信息
- 提供参考建议
- 执行自动化监控
- 记录投资笔记

⚠️ **投资决策需谨慎**，建议：
- 重要决策咨询持牌金融顾问
- 设置合理的止损线
- 不要依赖 AI 预测
